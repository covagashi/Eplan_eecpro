# Part 3 - Writing values into parameters by importing a CSV file

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Importing parameter values Importing parameter values Part 3 - Writing values into parameters by importing a CSV file The values that are stored in a file can be imported as parameter values into an existing configuration by means of the import wizard. The parameter values are assigned as key-value pairs either to individual parameters or to a parameter of the Map type. Depending on the scheme used the parameter values can be imported as a string or with their respective data type. Files in CSV or Excel format can be used to this purpose. This part shows the import using CSV files as an example.
