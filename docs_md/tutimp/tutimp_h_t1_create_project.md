# Part 1 - Creating a configuration by importing a file

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Creating configuration Creating configuration Part 1 - Creating a configuration by importing a file A new configuration is created by using the import wizard. Depending on the file type of the specified import file, the import wizard varies the input boxes for additionally required files, for example scheme and template file. Possible import files are CSV, Excel, IMX and XML files. The import scenarios are described using IMX and XML files as an example for this part.
