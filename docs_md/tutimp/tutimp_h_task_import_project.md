# Task: Creating a configuration by importing

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Task: Creating a configuration by importing Configurations can be created, as well as project components placed on placeholders and extension points, or parameter values be set, by importing IMX, XML, CSV or Excel files. All the imported file formats are converted internally into the IMX format. Default templates are, in part, used in the process. This tutorial shows how files of the formats IMX, XML and CSV are applied. These file formats are imported in Parts 1 to 3 by means of an import wizard:
     * Part 1: Creating a new configuration.
     * Part 2: Placing a project component on the suitable placeholder or extension point.
     * Part 3: Setting several parameter values.
Commands and actions are used in the following parts:
     * Part 4: A parameter of the Map type is filled with values.
A modified mechatronic modular system that is made available in the preparatory steps is required for this tutorial.
