# Chapter overview

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Chapter overview The whole EEC documentation consists of the following chapters:
     * Introduction
     * Concept
     * EPLAN Engineering Configuration
     * Tutorial
     * Reference
     * Administration
     * Release Notes
The chapter [Concepts](concept_k_concepts.htm) presents the methodological principles of EEC. In addition to the functional configuration methodology, it also describes the IT philosophy of EEC, concepts regarding team support, as well as other topics. The Concepts chapter does not require previous knowledge of EEC and is aimed at readers that do not themselves use EEC. The chapter [BASIC](eecbase_k_base.htm) uses workflows that are to be supported by the system to explain how the software works. The EEC implements the principles outlined in the Concepts chapter. As a result, the sections in the two chapters are closely correlated. Hyperlinks are used to create logical links between these sections. The [Tutorial](tutorial_k_tutorial.htm) chapter is designed to provide a brief introduction to the topic of functional configuration and makes no claim to be complete. The tutorial, however, uses hyperlinks to direct you to sections in the previous two chapters, where you can read more on the topics. The key tool to automate calculations and formally model rule sets within the system is the formula language. Graphical user interfaces can be created for project engineers with the aid of XML files. The [Reference](reference_r_reference.htm) chapter serves as a reference guide for the following:
     * Formula language.
     * Regular expressions.
     * User interfaces (Form-UI).
     * Scripting methods.
     * Text generator functions.
     * Commands.
     * 2D graphical editors (GRAPH2D).
     * Tags of Job Server job definitions.
     * Tags of IMX data format.
     * Tags for transformation of XML import files.
     * Tags for transformation of Excel import files.
The [Administration](admin_r_administration.htm) chapter examines the various installation options for the EEC, system requirements, etc., and, like the EEC chapter implements the concepts from chapter 2. Which new features are implemented in the current release is listed in the [Releases Notes](releasenotes_k_feature_releases.htm) chapter.
