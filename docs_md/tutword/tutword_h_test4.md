# Generating the current status of the component list

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Generating the current status of the component list To create and view the ListOfComponents, open the Mechatronic popup menu in the Project Catalog and select the SequenceSelectionAction Word > Generate and open Word document. The created ListOfComponents looks as follows: The Eplan logo is on the cover sheet and the symbols for actuator and sensor are shown in the tables. The table of contents is not up-to-date, although the contents that are shown appear to be correct. If the table of contents was updated before Chapter.docx was saved, the formula (*{LOOP:dos}*)(*{Content}*)(*{END_LOOP}*) is shown as chapter 1.1 as the chapter heading. The SequenceSelectionAction Update Word document fields is used to update the table of contents. Open the Mechatronic popup menu in the Project Catalog and select the SequenceSelectionAction Word > Update Word document fields.
