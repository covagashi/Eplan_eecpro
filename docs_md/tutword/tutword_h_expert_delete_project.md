# Deleting a project

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Deleting a project When modifying the mechatronic modular system, dependencies to the instances in the project can become invalid and result in severe model errors. Before changes are performed to a modular system, the project should be removed. [Removing the Feeder project](javascript:void\(0\);)
    1. Select the Feeder project in the project catalog.
    2. Click [Del].
    3. Click [Finish].
