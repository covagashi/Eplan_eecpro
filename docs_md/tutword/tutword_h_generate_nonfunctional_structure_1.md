# Creating parameters for plugs and sockets

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Creating parameters for plugs and sockets Before the plugs and sockets can be assigned to the Word components, these must first be created as parameters:
    1. Mark the Mechatronic.Word.Parameter unit in the T_Mechatronic_ModularSystem library.
    2. In the popup menu select New > Engineering > Parameter.
    3. Enter Plug as the name.
    4. Enter Plug as the type. The type can either be found by using the Finder  or be entered directly.
    5. Repeat steps 1 to 4 for the parameter with Socket as the name and type.
