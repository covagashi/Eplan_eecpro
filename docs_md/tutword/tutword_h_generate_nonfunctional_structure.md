# Generating non-functional structures

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Generating non-functional structures Hierarchical discipline structures, which are different not in that elements are omitted from the mechatronic structure, are defined using plug parameters and socket parameters. If a Word component does not have a plug parameter, it is subordinated in the mechatronic structure of the next discipline component the parent level that permits this through the internal definition of sublevels. If a Word component does have a plug parameter, it is placed below the discipline components that have a socket parameter with the same value as its plug parameter.
