# Adding body and chapter to mechatronic components

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Adding body and chapter to mechatronic components In the first step, you need to link the Word components for the body and chapter of the ListOfComponents with the mechatronic components:
    1. Double-click to open the component Feeder in the T_Mechatronic_ModularSystem library in the Stations unit.
    2. Using Drag & Drop, drag the Body component once, and the Chapter component twice into the component panel of the editor.
    3. Rename Body to ListOfComponents.
    4. Rename Chapter and Chapter2 into ActuatorsTable and SensorsTable.
Tip Switch to editor mode by clicking the selected component again or by pressing [F2].
