# Adding ImageObjects to the mechatronic model

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Adding ImageObjects to the mechatronic model The Eplan logo and, in the tables of the actuators and sensors in the first column, the symbols of the components are to be displayed on the cover sheet below the title. Instead of permanently installing these figures they are to be referenced by means of a formula. The GIF file Logo.gif () that is directly referenced serves as an image object for the Eplan logo. The images Actuator.gif() and Sensor.gif () previously specified should be used as the image objects for actuators and sensors. The image objects are assigned to the relevant type of the component in the image register. A formula can access this image register and determine the matching image.
