# Generating the current status of the component list

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Generating the current status of the component list Generate the current status of the component list. This operation again creates an error. The reason for this is that the component Image.Logo to be accessed in the model has not yet been created. The error message can be acknowledged by pressing [OK]. The created component list looks as follows: The logo on the cover sheet is, however, missing and the table of contents is not up-to-date, although the contents that are shown appear to be correct. If the table of contents was updated before Chapter.docx was saved, the formula (*{LOOP:dos}*)(*{Content}*)(*{END_LOOP}*) is shown as chapter 1.1 as the chapter heading. The table rows for actuators and sensors are now respectively contained in the correct chapter. The images of the actuators and sensors are, however, still missing in the Symbol column. Note By using formulas instead of predefined plug and socket values, it is possible to create any required discipline-specific structures.
