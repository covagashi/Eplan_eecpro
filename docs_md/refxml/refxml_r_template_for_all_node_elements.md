# Template for all elements of the node (Root Template)

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Template for all elements of the node (Root Template) The following rule has the effect that the current element is called up. It is used, for example, to call up the root node of the XML. [Example code](javascript:void\(0\);)
    
        <xsl:template match="/">
    	<!-- Content -->
    </xsl:template>
