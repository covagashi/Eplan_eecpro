# XML Transformation

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Import XML Transformation The purpose of the XML import is to create projects based on a modular system. The XML files are usually created by third-party systems and therefore must be converted during the import by XSL Transition into the IMX format that can be interpreted by EEC. The import of the IMX files commonly occurs through the import wizard, or an equivalent command. The XML file to be imported must meet the usual requirements for XML files.
