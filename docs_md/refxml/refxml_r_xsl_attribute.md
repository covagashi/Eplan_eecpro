# \u003cxsl:attribute\u003e

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  <xsl:attribute> XSLT The following rule creates an attribute name with the value Content in the target document. [Example code](javascript:void\(0\);)
    
        <xsl:attribute name="name">
    	<!-- Content -->
    </xsl:attribute>
