# Importing an architecture library

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Importing an architecture library To add a discipline component for the ListOfComponents for the mechatronic modular system T_Mechatronic_ModularSystem, the T_ListOfComponents_Architecture library has to be imported:
    1. Double-click to open the T_Mechatronic_ModularSystem library.
    2. Click Imported Libraries to switch to the corresponding editor page.
    3. Move the T_ListOfComponents_Architecture library to the Imported libraries diagram.
