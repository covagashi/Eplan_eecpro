# Creating the basic structure

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Creating the basic structure A number of objects are required to define a text discipline. They contain, for example, specifications on how parameters can be identified in text files or define how files are created. These required text discipline objects can be generated quickly by using the wizard:
    1. Call up the popup menu of the T_ListOfComponents_Architecture and select New > Text discipline....
    1. Enter the ListOfComponents name into the Text discipline name field.
    2. Click [Finish].
