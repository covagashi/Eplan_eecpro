# Generation of finished ListOfComponents

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Generation of finished ListOfComponents To create and view the ListOfComponents, open the Mechatronic popup menu in the Project catalog and select ListOfComponents > Generate structure and open: The actuators and sensors are all now contained in the correct chapter. By using formulas instead of predefined plug and socket values, it is possible to create any required discipline-specific structures.
