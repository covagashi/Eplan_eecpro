# Creating plug and socket parameters

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Creating plug and socket parameters Before the plugs and sockets can be assigned to the discipline components, these must first be created as parameters:
    1. Select the Parameter unit in T_Mechatronic_ModularSystem.
    2. Select New > Engineering > Parameter in the popup menu.
    3. Enter Plug in the Name field.
    4. EnterPlug as the type: The type can either be found by using the Finder  or be entered directly.
    5. Repeat steps 1 to 4 for the parameter with Socket as the name and type.
