# Mechatronic model: Generating non-functional structures

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Mechatronic model: Generating non-functional structures Hierarchical discipline structures, which are different not in that elements are omitted from the mechatronic structure, are defined using Plug parameters and Socket parameters. If a discipline component does not have a plug parameter, it is subordinated in the mechatronic structure of the next discipline components to the same or parent level that permits this through the definition of sublevels.
