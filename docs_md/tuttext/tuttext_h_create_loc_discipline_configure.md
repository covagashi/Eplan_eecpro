# Configuring a text discipline

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Configuring a text discipline We will now create the ListOfComponents in a single text file. This will be built from chapters that in turn consist of lines. To specify that all text blocks should be generated in one file, the text discipline has to be configured accordingly:
    1. Double-click to open the ListOfComponents text discipline configuration you just created.
    2. Select the image  (<EEC installation folder>\install\icon-collection\komponentenliste.GIF) with .
    3. Enter an optional comment for the text discipline.
    4. Open the Configuration editor page.
    1. Enter the path resources\Tutorial\ListOfComponents in the field Resource storage location.
    2. Save with [Ctrl] \+ [S] or .
