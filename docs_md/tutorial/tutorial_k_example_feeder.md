# Example Feeder

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Example Feeder The example machine is a feeder that removes parts from a magazine and moves them for further processing. After the parts are removed from the magazine, an optional check is performed to determine whether the workpiece is correctly positioned. After this has been performed, the part can optionally be rejected as a defective part.
