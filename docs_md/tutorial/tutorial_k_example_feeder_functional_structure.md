# Functional assembly

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Functional assembly The functional assembly describes the functions to be performed by the mechatronic components. The feeder is divided into the Insert, Inspect, Discard, Store, and Move function groups. Every function group in turn consists of several function units, e.g., Separator, Stack, or Axis. If you break down the function units even further into sensors and actuators, the functional assembly structure would look as follows: In the figure above, the composition structure is displayed by means of is contained in relationships and a line with a filled rhombus as the arrow tip. As you can see, smaller components in particular, for example, Cylinder, Valve, or position sensor, are installed multiple times.
