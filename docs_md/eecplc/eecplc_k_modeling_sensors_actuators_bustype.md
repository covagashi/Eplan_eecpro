## Bus type

The bus type for sensors and actuators is defined in a manner analogous to that described for control components in the section on [Modeling bus topology](eecplc_k_create_controller_topology_modeling_bustopology.htm). It should be noted that, in a departure from the actual technical conditions (e.g. direct wiring), the connection between bus terminal and sensor or actuator is modeled as a bus connection.
