## Specific actions for CoDeSys

The following commands are available for creating code and structure for CoDeSys:

    1. [CoDeSys.GenerateCoDeSysCodeCommand](refcommands_r_codesys_generatecodesyscodecommand.htm): Generates the program code for CoDeSys. The code can be viewed in the navigator.
    2. [CoDeSys.GenerateCoDeSysStructureCommand](refcommands_r_codesys_generatecodesysstructurecommand.htm): Generates the CoDeSys structure.
    3. [CoDeSys.GenerateENIProjectCommand](refcommands_r_codesys_generateeniprojectcommand.htm): Passes the created CoDeSys program code to the ENI server. This requires that an ENI Server must be reachable. The basic properties of the ENI server can be set in the model variables (Model > Model variables > Disciplines > PLC > CoDeSys > Code generator).
