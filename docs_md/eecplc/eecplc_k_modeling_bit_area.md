## Bit area

The parameter for the bit area defines how many bits it takes for an overflow to occur. Frequently, the parameter for the bit area has a value of 8 for byte-based addressing or 16 for word-based addressing.

The name of the parameter for the bit area can be changed via the (Model > Model variables menu) Disciplines > IO generator > Name of the parameter for the bit area model variable.
