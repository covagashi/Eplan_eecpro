## Specific actions for AutomationWorX

The following commands are available for creating code and structure for AutomationWorX:

    1. [AutomationWorX.GenerateAutomationWorXCodeCommand](refcommands_r_automationworx_generateautomationworxcodecommand.htm): Generates the program code for AutomationWorX. The code can be viewed in the navigator.
    2. [AutomationWorX.GenerateAutomationWorXStructureCommand](refcommands_r_automationworx_generateautomationworxstructurecommand.htm): Generates the AutomationWorX structure.
