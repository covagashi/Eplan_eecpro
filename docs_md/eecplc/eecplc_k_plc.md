## PLC

The PLC discipline enables modeling tasks involving all types of control software to be performed. On the one hand, this discipline provides its own basic objects, methods and actions for general modeling, while on the other, it serves as a basis for more in-depth integration of specific PLC systems, such as STEP 7, CoDeSys, AutomationWorX, etc.

The basic technology that underlies resource design is similar to that used in the text discipline. However, in many areas it exhibits distinctive differences stemming primarily from the specifics of the documents in use.
