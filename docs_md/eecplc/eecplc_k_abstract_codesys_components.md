## Abstract CoDeSys components

You define a new superclass with an abstract component. The new superclass inherits its properties (e.g., parameters) to all classes derived from it. One essential property of an abstract component is that no resource is allocated to it. Only an actual component derived from the abstract component contains a resource.

### Note

You cannot undo the abstract option.
