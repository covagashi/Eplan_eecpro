## Specific actions for STEP 7

The following commands can be used to create projects, code, structure and function-block numbers for STEP 7:

    1. [Step7.GenerateSimaticProjectCommand](refcommands_r_step7_generatesimaticprojectcommand.htm): Passes the created Simatic program code to the Simatic-Manager. The Simatic-Manager must be installed on the workstation. In the settings the SIMATIC project in one source option under Disciplines > PLC > STEP 7 > SIMATIC is to be marked. If the option is not marked, only the marked sources are compiled and replaced in the Simatic-Manager.
    2. [Step7.GenerateStep7CodeCommand](refcommands_r_step7_generatestep7codecommand.htm): Generates the STEP 7 program code.
    3. [Step7.GenerateStep7NumbersCommand](refcommands_r_step7_generatestep7numberscommand.htm): Generates the block numbers for STEP 7 blocks.
    4. [Step7.GenerateStep7StructureCommand](refcommands_r_step7_generatestep7structurecommand.htm): Generates the STEP 7 structure.
