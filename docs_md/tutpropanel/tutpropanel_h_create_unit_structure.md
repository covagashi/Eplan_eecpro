# Creating a unit structure

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Creating a unit structure The discipline components and their parameters should be created in a single structure. The structure is built of hierarchically ordered units. In the T_Mechatronic_ModularSystem library create the ECAD unit, and in it, the ProPanel subunit:  
    1. Select the T_Mechatronic_ModularSystem library.
    2. Select New > Engineering > Unit in the popup menu.
    3. In the Name field, enter the text ECAD.
    4. Save the unit with [Ctrl] \+ [S].
    5. Select the ECAD unit.
    6. Repeat steps 2 to 4. Enter the unit name ProPanel in the process.
