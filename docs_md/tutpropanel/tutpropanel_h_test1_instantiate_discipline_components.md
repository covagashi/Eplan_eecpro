# Instantiating Eplan Pro Panel discipline components

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Instantiating Eplan Pro Panel discipline components As with the mechatronic components, the Eplan Pro Panel components can also be instantiated. This is done by dragging and dropping them from the library catalog view into the project tree, where they will appear under the ECAD discipline root node:
    1. In the Project catalog open the Feeder project.
    2. Drag and drop the library component Schematic onto the project library Feeder.
    3. The library component Schematic is initiated as a root component of the ECAD discipline.
If you place a library component somewhere other than the permitted position, the cursor will change to a stop sign to indicate this.
    1. Drag the LayoutSpace library component onto the Schematic root component.
    2. One after the other drag the library components AE1050_500, AE1050_500_MountingPanel, MP_Plane_Front, KK3040, TS35_7_5 and Terminal_PxC_3031212 into the Project Explorer.
    3. The discipline components of the modular system can be correctly instantiated according to the hierarchy.
