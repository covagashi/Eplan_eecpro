# Creating an enclosure for Feeder with Eplan Pro Panel

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Creating an enclosure for Feeder with Eplan Pro Panel The task is to create an enclosure with Eplan Pro Panel from the Feeder functional, mechatronic configuration. To accomplish this, the T_Mechatronic_ModularSystem will be expanded to include the pre-built Pro Panel library. Next, the Pro Panel components will be assigned to the components of the functional, mechatronic configuration. Finally, we will show how to create the enclosure.
