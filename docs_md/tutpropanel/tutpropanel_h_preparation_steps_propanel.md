# Preparatory steps in Pro Panel

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Preparatory steps in Pro Panel Parts data that might not be available in your parts database is necessary for the tutorial. Therefore all used parts are made available as EDZ files in order to import them. [Importing parts data as EDZ files](javascript:void\(0\);) The parts data for this tutorial are stored as EDZ files in this folder:
    
        <EEC installation folder>\install\Tutorial\ProPanel

This is how you import the parts data into your database:
    1. Start Eplan Pro Panel or Eplan Electric P8 with the add-on for Pro Panel.
    2. Select the following commands: Tab Master data > Command group Parts > Management.
    3. Click [Extras].
    4. Select Import.
The Import data sets dialog opens.
    1. For the File name field select the EDZ files from the folder specified above. To do this use the [...] button.
    2. Confirm with [Open].
    3. If necessary, mark the Only add new records option.
    4. Confirm with [OK].
Then you can close Eplan Pro Panel.
