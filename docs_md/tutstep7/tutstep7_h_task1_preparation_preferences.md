# Setting settings for the STEP 7 discipline

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Setting settings for the STEP 7 discipline In order for the EC to access the blocks that were unzipped and modified in the previous step, it is necessary to first enter the path of these files under the settings:
    1. Select Window > Settings.
The Settings (Preferences) window now appears:
    1. Select Disciplines > PLC > STEP 7 in the tree structure.
    2. Enter the path of the blocks that you want to unzip in the Source folder field.
    
        \resources\Tutorial\Step7

    1. Confirm with [OK].
Note Because this path has been changed, the STEP 7 discipline no longer works in the demo model! To work with the demo model, press [Restore default values] to restore the previous path.
