# Creating parameters for model variables

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Creating parameters for model variables Parameters must now be created for model variables for the parameters names defined in the [Defining parameter names for model variables](tutstep7_h_task2_define_parametername.htm) section:
    1. Select the Parameter unit.
    2. Select New > Parameter in the shortcut menu.
    3. In the Name field, enter the text IO_BitRange. The preset type String is set.
    4. Save the parameter.
    5. Repeat steps 3 and 4 for the following parameters (all with type String):
