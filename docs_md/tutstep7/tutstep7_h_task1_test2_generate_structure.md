# Test 2 - Generating instead of manually creating the current STEP 7 structure

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Test 2 - Generating instead of manually creating the current STEP 7 structure The structure should not be created manually but created.
    1. Select Mechatronic in the Project Catalog in the Feeder project.
    1. Select STEP 7 > Generate structure in the popup menu.
The same structure is created as above during manual instantiation. The only difference is the FB_Insert, FB_Inspect, and FB_Move function blocks are not yet contained in the structure. This will be done in the next step.
