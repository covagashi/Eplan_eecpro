# Inserting and configuring the Caller parameter in FR_SymbolTable and FR_SymbolsGeneral

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Inserting and configuring the Caller parameter in FR_SymbolTable and FR_SymbolsGeneral The fragments FR_SymbolTable and FR_SymbolsGeneral also require the parameter Caller to be integrated in the discipline structure by means of the Plug-Socket principle. Inserting the Caller parameter:
    1. Open the fragment FR_SymbolTable.
    2. Select the Parameters editor.
    3. Drag the Caller parameter from the Parameter unit into the editor.
    4. For the Caller parameter, enter the value SymbolTable.
    5. Repeat steps 1 to 4 for the fragment FR_SymbolsGeneral.
