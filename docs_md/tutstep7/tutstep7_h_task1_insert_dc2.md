# Adding discipline components to mechatronic components

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Adding discipline components to mechatronic components The FB_Insert, FB_Inspect, and FB_Move function blocks still have to be inserted into the corresponding mechatronic components.
    1. Open the Insert function group in T_Mechatronic_ModularSystem.
    2. Insert the FB_Sequence function block.
    3. Open the Inspect function group.
    4. Insert the FB_Inspect function block.
    5. Open the Move function group.
    6. Insert the FB_Move function block.
