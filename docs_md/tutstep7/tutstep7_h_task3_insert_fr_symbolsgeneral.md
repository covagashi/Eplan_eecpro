# Installing and configuring the fragment FR_SymbolsGeneral in the Feeder Station

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Installing and configuring the fragment FR_SymbolsGeneral in the Feeder Station Analogous to the "Installing the configuring the fragment FR_SymbolTable in actuators and sensors" section, the fragment FR_SymbolsGeneral is now installed into the Feeder Station and the parameters are provided with values.  
    1. Open the Feeder station.
    2. Drag the fragment FR_SymbolsGeneral from the Step7 unit into the component editor.
    3. Select the installed fragment FR_SymbolsGeneral.
    4. Enter the following formulas and values for the parameters:
Parameter | Value  
---|---  
Bit0 | =mc.$Bit0_IX  
Bit1 | =mc.$Bit1_IX  
Bit2 | =mc.$Bit2_IX  
Bit3 | =mc.$Bit3_IX  
Bit4 | =mc.$Bit4_IX  
Bit5 | =mc.$Bit5_IX  
Bit6 | =mc.$Bit6_IX  
Bit7 | =mc.$Bit7_IX  
Bit8 | =mc.$Bit8_IX
