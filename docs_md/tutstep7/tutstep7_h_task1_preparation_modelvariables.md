# Configuring model variables

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Configuring model variables The Instance Name Parameter For Discipline Structure model variable is used to create the program code. The value of a model variable is passed at runtime by a parameter with the same name. The value of the model variable is specified as follows:
    1. Select Model > Model Variables to open the model variables.
    1. Navigate to the model variable Disciplines > Instance Name Parameter For Discipline Structures.
    2. Enter the value DisciplineName.
    3. Save ([Ctrl] \+ [S]) and close the model variables.
