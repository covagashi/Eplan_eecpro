# Installing the fragments FR_Error_Dec and FR_Error_NW in Axis and Separator Function units

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Installing the fragments FR_Error_Dec and FR_Error_NW in Axis and Separator Function units The fragments FR_Error_Dec and FR_Error_NW must now be installed and configured in the Axis and Separator Function units.
    1. Open the Axis Function unit.
    2. Drag the fragments FR_Error_Dec and FR_Error_NW from the Step7 unit into the component editor.
    3. Repeat steps 1 and 2 for the Separator Function unit.
