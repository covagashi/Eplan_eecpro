# Installing the SiemensCPU controller in the mechatronic component

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Installing the SiemensCPU controller in the mechatronic component The SiemensCPU discipline component is installed in the S7_313C mechatronic component:
    1. Open the S7_313C component.
    1. Drag the SiemensCPU discipline component from the library catalog to the component editor.
