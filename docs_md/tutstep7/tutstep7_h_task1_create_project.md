# Creating the Simatic project in the project explorer

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Creating the Simatic project in the project explorer The Simatic Manager is required to create the Simatic project in the project explorer.
    1. Open the project explorer view (Window > Show view > Project explorer).
    1. Select the Feeder.
    2. Select Create Simatic Project in the shortcut menu.
Note In order to be able to create the SIMATIC project, the option Single source Simatic project have to be disabled in the settings (see [SIMATIC](admin_r_prefs_plc_s7simatic.htm)).
