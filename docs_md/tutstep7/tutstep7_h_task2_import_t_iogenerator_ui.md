# Importing T_IOGenerator_Ui library into mechatronic modules

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Importing T_IOGenerator_Ui library into mechatronic modules To add the functions for the IO_Generator to the T_Mechatronic_ModularSystem model, you will need to import the T_IOGenerator_Ui library.  
    1. Double-click to open the T_Mechatronic_ModularSystem library.
    2. Select the Imported libraries editor.
    3. Click .
The library finder now appears:
    1. Click [Search].
    2. Select the T_IOGenerator_Ui library.
    3. Confirm with [OK].
