# Changing the level configuration for a station

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Changing the level configuration for a station For the future configuration, the level configuration of the Station component must be changed, so that a Function unit can be inserted below the Station in addition to the Function group. The level configuration for the Station component is changed as follows:
    1. Open the Station architecture component.
    2. Open the Level configuration editor.
    3. Click .
The Finder dialog is opened:
    1. Press the [Enter] button to list all available components.
    2. Select the Function unit component in the Finder.
    3. Confirm with [OK].
