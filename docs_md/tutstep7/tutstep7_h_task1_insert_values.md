# Enter values for the DBNumber parameter

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Enter values for the DBNumber parameter The task is to assign a value for the DBNumber parameter:
    1. Open the FB_Sequence function block.
    2. Select the Parameters editor.
    3. Enter 1 as the value for the DBNumber parameter.
    4. Repeat steps 1 and 3 for the FB_Insert, FB_Move and FB_Inspect function blocks. Enter the values from the following table for the DBNumber parameter.
Function block | DBNumber | Comment  
---|---|---  
FB_Sequence | 1 |   
FB_Insert | 9999 | Multi-instance  
FB_Move | 9999 | Multi-instance  
FB_Inspect | 9999 | Multi-instance
