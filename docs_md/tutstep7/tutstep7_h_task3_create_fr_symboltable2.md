# Creating new fragments FR_SymbolTable and FR_SymbolsGeneral

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Creating new fragments FR_SymbolTable and FR_SymbolsGeneral The FR_SymbolTable and FR_SymbolsGeneral fragments will provide the required contents at a later time when the symbol table is created. First the fragments are created and then installed and parameterized in the actuators and sensors in additional steps. Create the fragment FR_SymbolTable in unit Step7:
    1. Select the Step7 unit.
    2. From the popup menu, select New > STEP 7 > Source > Fragment.
    3. In the Resource location panel, click Import.
    4. Select the file FR_SymbolTable.sdf in the following dialog.
    5. Confirm with [Open].
Create the fragment FR_SymbolsGeneral in unit Step7:
    1. Select the Step7 unit.
    2. From the popup menu, select New > STEP 7 > Source > Fragment.
    3. In the Resource location panel, click Import.
    4. Select the file FR_SymbolsGeneral.sdf in the following dialog.
    5. Confirm with [Open].
