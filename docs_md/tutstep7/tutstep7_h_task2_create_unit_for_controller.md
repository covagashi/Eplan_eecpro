# Creating the ControllerComponents unit

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Creating the ControllerComponents unit For the control, a mechatronic component is created that is to be stored in a separate unit.  
    1. Select the T_Mechatronic_ModularSystem library.
    2. Select New > Unit in the shortcut menu.
    3. In the Name field, enter the text ControllerComponents.
    4. Save the unit.
