# Creating parameters for input and output addresses

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Creating parameters for input and output addresses Parameters of type IX and OX must be created for the input and output addresses.
    1. Select the Parameter unit.
    2. Select New > Parameter in the shortcut menu.
    3. In the Name field, enter the text Bit0_IX.
    4. In the Type field, enter the text IX.
    5. Save the parameter.
    6. Repeat steps 2 to 5 for the parameters with names Bit1_IX to Bit23_IX (all with type IX).
    7. Repeat steps 2 to 5 for the parameters with names Bit0_QX to Bit15_QX and enter the type QX for these parameters.
