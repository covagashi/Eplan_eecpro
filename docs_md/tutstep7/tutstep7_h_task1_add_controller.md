# Adding a controller

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Adding a controller A controller is first needed as the root element. This must be created in the new Step7 unit.
    1. Select the Step7 unit.
    1. From the popup menu, select New > STEP 7 > Control Device > Controller.
    2. In the Name field, enter the text SiemensCPU.
    3. Save the controller ([Ctrl] \+ [S]).
