# Allocating the parameters of the S7_313C component

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Allocating the parameters of the S7_313C component For creating the input and output addresses (IO_Generation), the S7_313C component must first allocate the following parameters:
    1. Open the S7_313C component.
    2. Open the Parameter editor.
    3. Drag the following parameters from the Parameter unit into the editor.
     * IO_BitRange
     * IO_InputBlockSize
     * IO_InputStartAddress
     * IO_InputStartAddressFormat
     * IO_OutgoingBus
     * IO_OutputBlockSize
     * IO_OutputStartAddress
     * IO_OutputStartAddressFormat
     * IO_Place
     * IO_Places
     * Bit0_IX to Bit23_IX
     * Bit0_QX to Bit15_QX
