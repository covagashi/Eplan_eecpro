# Installing the S7_313C ControllerComponent in Feeder Station

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Installing the S7_313C ControllerComponent in Feeder Station Previously the SiemensCPU controller was installed directly in the Feeder station (see [Adding discipline components to mechatronic components](tutstep7_h_task1_insert_dc1.htm)). Instead of the SiemensCPU controller, now the S7_313C mechatronic component is installed in the Feeder station. The SiemensCPU controller is first uninstalled. Uninstalling the SiemensCPU controller:  
    1. Open the Feeder station.
    2. Select the SiemensCPU controller in the component editor.
    3. Click on the Delete component symbol or press the [Del] button.
Installing the S7_313C component in the Feeder station
    1. Open the Feeder station.
    2. Drag the S7_313C component into the component editor.
