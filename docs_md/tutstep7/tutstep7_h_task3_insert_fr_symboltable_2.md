# Installing and configuring the fragment FR_SymbolTable in the Stack Function unit

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Installing and configuring the fragment FR_SymbolTable in the Stack Function unit Analogous to the Installing the configuring the fragment FR_SymbolTable in actuators and sensors section, the fragment is now installed into the Stack Function unit and the parameters are provided with values.  
    1. Open the Stack function unit.
    2. Drag the fragment FR_SymbolTable from the Step7 unit into the component editor.
    3. Select the installed fragment FR_SymbolTable.
    4. Enter the following formulas and values for the parameters:
Parameter | Value  
---|---  
Symbol | F_Pulse_PartInStack  
Address | M13.1  
DataType | BOOL  
Comment | Pulse Part in Stack
