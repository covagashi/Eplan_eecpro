# Creating Actuator and Sensor parameters

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Creating Actuator and Sensor parameters Lists of actuators and sensors that are accessed in the formulas of the previous steps are created with the new Actuator and Sensor parameters to be created:
    1. Select the Step7.Parameter unit.
    2. Select New > Parameter in the shortcut menu.
    3. In the Name field, enter the text Actuator.
    4. In the Type field, enter the text List.
    5. Repeat steps 2 to 4 for the Sensor actuator.
