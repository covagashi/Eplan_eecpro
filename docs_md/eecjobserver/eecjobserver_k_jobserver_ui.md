##  User interface of Job Server

A user interface is available for Job control and administration of the Job Server. This can be displayed both in EEC and in a browser. When the Job Server is started, the Job Server perspective is opened automatically. For information on opening the Job Server perspective manually see [Perspectives](eecbase_k_perspectives.htm).

The URL specified in the address line is used for display in a browser.

The user interface provides the categories Workers, Logs and Jobs in the navigation bar.
