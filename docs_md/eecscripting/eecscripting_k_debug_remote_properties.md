# Settings in Eclipse

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Settings in Eclipse The following settings have to be set for the debugging of a newly developed plug-in against an installed EEC. Requirements
     * Eclipse is installed.
     * The settings are open.
[Specifying an installed JRE](javascript:void\(0\);)
    1. Navigate to Java > Installed JREs.
    2. Add the JRE of a compatible version to the list.
    3. Mark the compatible version.
[Specifying the target platform](javascript:void\(0\);)
    1. Navigate to Plug-in Development > Target Platform.
    2. Add an installed EEC to the Target Platform list.
    3. Select the installed EEC in the list.
    4. The installed EEC is now (Active).
