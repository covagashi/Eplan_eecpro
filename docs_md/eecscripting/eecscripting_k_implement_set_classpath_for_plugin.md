# Specifying a class path for the plug-in

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Specifying a class path for the plug-in The class path has to be set so that the classes of the new plug-in become generally visible and thus applicable.
    1. Open the MANIFEST.MF file of the new plug-in.
    2. Open the Runtime tab.
    3. Create a new entry in the Classpath list.
    4. Enter a period "." in the Enter new library name below field.
    5. Confirm with [OK].
