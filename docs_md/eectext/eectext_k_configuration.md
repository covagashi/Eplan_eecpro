# Configuration of text disciplines

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Configuration of text disciplines Previously, the text discipline was set up and configured with the help of a special view. On the basis of these configurations, the system schema for EEC was extended to make the text discipline available for modular-system development and for configuration. Analogous to the requirements for mechatronic level configurations, text disciplines were configured in libraries.
