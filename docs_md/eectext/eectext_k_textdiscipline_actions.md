# Text discipline actions

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Text discipline actions In order to create text discipline documents in the workspace of EEC and to open them with an editor, actions are executed via the context menu to initiate the following commands:  
    1. [TextDiscipline.GenerateTextDisciplineCodeCommand](refcommands_r_textdiscipline_generatetextdisciplinecodecommand.htm): Generates the text discipline files. They can be viewed in the navigator.
    2. [TextDiscipline.OpenTextDisciplineComponentsWithEclipseCommand](refcommands_r_textdiscipline_opentextdisciplinecomponentswitheclipsecommand.htm): Opens the text discipline component with EEC-internal editor (see also Editing resources directly from EC).
    3. [TextDiscipline.OpenTextDisciplineComponentsWithOSCommand](refcommands_r_textdiscipline_opentextdisciplinecomponentswithsocommand.htm): Opens the text discipline component in the editor which is assigned in the operating system to open text files.
