# Defining settings

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Defining settings Requirements:
     * EEC as a Job Server with Worker installation Version 2.5.1 or higher.
Start the Job Server.
    1. Enter the file <EEC Installation Folder>\eox\model.eox in the dialog Choose new backend.
    2. Deactivate the Open read-only option.
    3. Open the settings under Window > Settings.
    4. Navigate to Job Server > Job Server Worker.
    5. Enter \resources\Jobserver\jobs in the JMX file folder field.
    6. Enter the value 1 in the Maximum number of parallel Jobs field.
