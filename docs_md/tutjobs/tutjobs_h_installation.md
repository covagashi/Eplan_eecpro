# Job Server Installation

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Job Server Installation A Job Server installation is based on the stand-alone installation that is extended by means of plug-ins. Furthermore, additional entries are required in the initialization files and settings have to be set in the settings. In order to install the Job Server first carry out the steps that are listed in the section [Job Server with Job Server Worker installation](admin_h_installation_jobserver_worker.htm). Continue with the steps in the section [Preparatory steps on a file level](tutjobs_h_preparation_steps_files.htm).
