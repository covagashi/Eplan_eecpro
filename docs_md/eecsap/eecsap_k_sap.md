## SAP interface

By means of the generic SAP interface data between SAP and EEC are transferred bidirectionally. On one hand, the function modules contained in the SAP standard can be activated. On the other hand, however, individually programmed modules can also be used.

The interface communicates between EEC and SAP by means of the JCo Java connector provided by SAP.

The EEC version 2.4 supports the SAP interface of version 1.0.
