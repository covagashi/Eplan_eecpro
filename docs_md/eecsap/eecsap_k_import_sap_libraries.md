## Import of SAP libraries

The SAP libraries provide the required library components and commands for using the SAP interface.

The libraries are provided for importing in the EOX file in the folder <EEC installation folder>\install\sap. The file name has the following syntax: sap_<Version number>.eox.

     * SAPToolkit
