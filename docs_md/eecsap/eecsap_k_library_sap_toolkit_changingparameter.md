## Parameter ChangingParameter

The ChangingParameter parameter contains individual parameters to be passed.

Syntax:
    
        =Map{
    	Pair {'EEC_CONFIG_KEY', List{
    		List{ 'MANDT', 'CONFIG_ID' },
    		List{ '100', '0050569E4B5D1ED58B89E46D2CE34F8C' }
    	} }
    }
