## Component SapFunctionBlockTypeX

The SapFunctionBlockTypeX component represents the actual SAP callup and is installed (pre-instantiated) in the SapObject component.

Here, the component ideally receives the name of the calling SAP module. It contains the following parameters by default:

     * FunctionBlock
     * ImportParameter
     * ImportTable
     * ExportParamFilter
     * ExportTableFilter
     * ChangingParameter
     * Result
