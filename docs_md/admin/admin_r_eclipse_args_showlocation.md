## -showlocation

Eclipse argument |  Values |  Usage  
---|---|---  
-showlocation |  - |  Optional  
**Annotation**  
Shows the path to the workspace in the title bar of the window. The argument can be extended to specify an alias. The alias is also displayed in the title bar of the window. Example:
    
        -showLocation
    "My EEC Workspace"
