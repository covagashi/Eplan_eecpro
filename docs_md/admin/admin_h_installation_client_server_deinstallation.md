## Uninstalling

To uninstall EEC from the client, proceed as follows:

Click Start > All Programs > Eplan > Engineering Configuration > Build-ID > Uninstall Eplan Engineering Configuration...

Runtime data created at the start must be deleted manually.

The files on the server must be uninstalled by deleting them manually.
