# Default comment

This functionality is only available for certain module packages. [Info / Copyright](license.htm)

You are here: 

## Default comment

Path to model variable:

Disciplines

Specify a string value. The value determines a text that is used for all comments without content.

The comment is merely displayed for each object in the comment field. Once the comment for an object has been changed, it can no longer be overwritten by the default comment. Thus, a change of the default comment will only affect all the empty comments.

The value of the default comment can be created using a simple text editor (e.g., Edit) and pasted in the Value column via the clipboard. Multiline comments can be pasted only via the key combination [Ctrl] \+ [V].
