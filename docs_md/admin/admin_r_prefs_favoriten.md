## Preferences for favorites

Path to the preferences:

Menu Window > Preferences

Dialog Preferences > Favorites

     * Use favorite system libraries activates the function so that all marked system libraries are listed as favorites in the New library dialog in the Favorites category. This way the favorites can be marked together with one click in the New library dialog and imported into the new library.
