## Parameter Name For The Functional Assignment

Path to model variable:

Disciplines > ECAD > Eplan Electric P8

Specify the name of a parameter. Add this parameter to the P8 discipline components. The value of the parameter in the discipline components defines the structure identifier for the functional relationship between higher-level function, plant sections and devices.
