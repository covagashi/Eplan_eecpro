# Restoring settings

This functionality is only available for certain module packages. [Info / Copyright](license.htm)  
  
You are here: 

## Restoring settings

Proceed as follows to restore settings:

  1. Start EEC
  2. Click File > Import....

The Import dialog opens.

  3. Navigate to General > Settings.
  4. Click Next >.

The Import settings dialog opens:

  5. Select the setting file previously exported.
  6. Select the Import all option.
  7. Click Finish.



Subsequently the settings of the previous installation are available again.
