# Restoring resources

This functionality is only available for certain module packages. [Info / Copyright](license.htm)

You are here: 

## Restoring resources

Resources previously backed up via a compression program are to be made available again. The resources can be stored, in a discipline-specific manner, in various folders.

Proceed as follows:

  1. If necessary, delete the resource folders.
  2. Select the zipped file.
  3. Click 7-Zip > Extract files... in the context menu.
  4. Select whether to restore the directory structure, and by which method.

Full pathnames restores the files with absolute path information.

Relative pathnames restores files with folders based on the folder that contains the zip file.

No pathnames restores files in the folder that contains the zip file.

  5. Confirm your selection with [OK].



The progress is shown during the extraction. Subsequently, the restored resources will be available.
