# Recommended configuration of the server

This functionality is only available for certain module packages. [Info / Copyright](license.htm)

You are here: 

This functionality described here is only available at certain configuration levels.

## Recommended configuration of the server

A computer used as a file server and / or database server should have at least the following properties:

Property | Requirement  
---|---  
**Processor speed** | See [Recommended configuration of the workstation](admin_r_recommended_configuration_workstation.htm)  
**RAM** | See [Recommended configuration of the workstation](admin_r_recommended_configuration_workstation.htm)  
**Hard disk** | See [Recommended configuration of the workstation](admin_r_recommended_configuration_workstation.htm); however, the hard disk on which the DBMS is installed should be NTFS-formatted.  
**Network card** | See [Recommended configuration of the workstation](admin_r_recommended_configuration_workstation.htm)
