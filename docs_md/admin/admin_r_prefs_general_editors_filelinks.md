This functionality is only available for certain module packages. [Info / Copyright](license.htm)

You are here: 

## File assignments

Path to the preferences:

Menu Window > Preferences

Dialog Preferences > General > Editors > File assignments

The default application of a specific editor for a specific file type can be defined by adding a file extension and assigning an editor. Additional editors can be added.
