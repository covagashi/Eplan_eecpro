## SIMATIC

Path to the preferences:

Menu Window > Preferences

Dialog Preferences > Disciplines > PLC > STEP 7 > SIMATIC

     * The SIMATIC project folder specifies the folder, where the SIMATIC Manager manages its projects. The SIMATIC default folder is used, if no project folder is specified.
     * The option SIMATIC project in one source must be enabled, if all source files for the STEP 7 project are located in a packed file. Ordinarily this option must be disabled to create STEP 7 files out of distributed resources. This option is to be selected so that the Compile SIMATIC sources action can be executed free of faults. The Compile SIMATIC sources action can only be used with a 32-bit installation of the EEC that is older than Version 2.8.
     * The number of files which may result in compiler errors without interrupting the STEP 7 file generation is specified in the Maximum number of files with compiler errors field.
