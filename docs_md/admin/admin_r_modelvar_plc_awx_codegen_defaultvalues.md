# Generate default values in calls

This functionality is only available for certain module packages. [Info / Copyright](license.htm)

You are here: 

## Generate default values in calls

Path to model variable:

Disciplines > PLC > AutomationWorX > Code Generator

Select a Boolean value. The value determines whether variables with default values are also transferred during the generation of block calls.

Valid values are true and false.
