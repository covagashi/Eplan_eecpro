## Write Macro Parameter Structure In Parameter

Path to model variable:

Disciplines > ECAD > Eplan Electric P8

Select a Boolean value. The value specifies whether a parameter whose name is specified with the model variable Parameter Name For The Macro Parameter Structure is created automatically when a P8 macro is read in.

Valid values are true and false.
