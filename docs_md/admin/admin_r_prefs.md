## Preferences

Path to the preferences:

Menu Window > Preferences

Dialog Preferences

Preferences are used to control the appearance and behavior of the user interface. All preferences are default preferences that apply to most users.

The preferences of the category displayed are reset to the default values by using the [Restore default values] button.

The modifications become active immediately when the [Apply] button is clicked.
