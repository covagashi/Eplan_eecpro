# EPAS configuration file name

This functionality is only available for certain module packages. [Info / Copyright](license.htm)

You are here: 

## EPAS configuration file name

Path to model variable:

Disciplines > PLC > CoDeSys > Elau

Specify a string value. The value determines the name of an EPAS configuration file. The specified file is written during the generation of the PLC code.
