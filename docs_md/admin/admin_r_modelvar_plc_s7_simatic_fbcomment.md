## Parameter Name For The FB Comment

Path to model variable:

Disciplines > PLC > STEP 7 > SIMATIC

Specify the name of a parameter for this model variable. Add this parameter to the SIMATIC discipline components. The values in the discipline components define the comments on the function blocks in the symbol table.
