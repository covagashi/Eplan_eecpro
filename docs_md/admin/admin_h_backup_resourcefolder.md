This functionality is only available for certain module packages. [Info / Copyright](license.htm)

## Backing up resources folders

You can back-up the resource folder with any compression software.

Since a separate resource folder can be specified for each discipline, several folders may have to be backed up.

The example describes compression using 7-Zip.

  1. Select the resource folder.
  2. In the popup menu click 7-Zip > Add to archive....
  3. Enter a file name for the zipped file, and confirm with [OK].
