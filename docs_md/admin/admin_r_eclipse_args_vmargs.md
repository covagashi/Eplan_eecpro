## -vmargs

Eclipse argument | Values | Usage  
---|---|---  
-vmargs | - | Mandatory  
**Annotation**  
This Eclipse parameter specifies that arguments for the Java Virtual Machine can be added afterwards. This entry is always mandatory, because EEC argument org.foederal.sn.connectionURL must always be stated.
