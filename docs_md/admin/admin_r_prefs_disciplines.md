## Preferences for disciplines

Path to the preferences:

Menu Window > Preferences

Dialog Preferences > Disciplines

     * The Root folder for notes specifies which folder is initially searched for note resources.

Default value: <EEC installation folder>\resources\Notes

The absolute path is shown below the input field.

     * The Temporary project name specifies the folder where temporary results are stored when project data is being created.
