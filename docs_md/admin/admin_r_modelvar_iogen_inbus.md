## Parameter Name For The Incoming Bus Line

Path to model variable:

Disciplines > IO Generator

Specify the name of a parameter. Add this parameter to the PLC discipline components. The string value of the parameter in the discipline components defines the name of the incoming bus. If the discipline component also has a connection point for an outgoing bus, the model variable [Parameter Name For The Outgoing Bus Line](admin_r_modelvar_iogen_outbus.htm) has to be specified as well.
