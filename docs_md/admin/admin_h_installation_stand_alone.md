# Stand-Alone installation

This functionality is only available for certain module packages. [Info / Copyright](license.htm)

You are here: 

## Stand-Alone installation

The installation is carried out as follows:

  1. Start the installation file.
  2. Select the language of the installation wizard.
  3. Click [OK].
  4. Acknowledge the following information page with [Next >].
  5. Accept the license agreement.
  6. Click [Next >].
  7. Select the installation variant Stand-alone installation.
  8. Click [Next >].
  9. Enter the target folder for the installation.
  10. Click [Next >].
  11. Indicate whether an entry is to be created in the start menu and/or desktop symbol.
  12. Click [Next >].
  13. Complete the installation overview with [Install].



Installation with database data management

### Note

This option is discontinued and cannot be used for new projects.

The documentation for the database data management is only available in the versions up to Version 2024.

  1. Open a Windows console.
  2. Navigate to the EEC installation file.
  3. Start the installation routine with the following command:



setup.exe /show-deprecated-features

  1. Select the language of the installation wizard.
  2. Click [OK].
  3. Acknowledge the following information page with [Next >].
  4. Accept the license agreement.
  5. Click [Next >].
  6. Select the installation variant Stand-alone installation.
  7. Click [Next >].
  8. Enter the target folder for the installation.
  9. Click [Next >].
  10. Indicate whether an entry is to be created in the start menu and/or desktop symbol.
  11. Click [Next >].
  12. Complete the installation overview with [Install].
  13. After the successful installation the administration tool for the database is available.



### Note

A corresponding license is required for operation with database data management.
