## Unit for Angles

Path to model variable:

Disciplines > ECAD >EPLAN Pro Panel

The model variable Unit for angles is used to specify whether angle specifications are interpreted in degrees or radians.

The value of the parameter is selected from a drop-down list. Valid values are Degrees and Radians.
