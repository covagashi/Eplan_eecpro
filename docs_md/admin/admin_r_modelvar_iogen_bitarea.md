## Name of the parameter for the bit area

Path to model variable:

Disciplines > IO Generator

Specify the name of a parameter. Add this parameter to the PLC discipline components. The value of the parameter in the discipline components defines after how many bits an overflow occurs. Usual specifications for the bit area are the values 8 for byte-wise addressing or 16 for word-wise addressing.
