## Parameter Name For The Window Macro Variant

Path to model variable:

Disciplines > ECAD > Eplan Electric P8

Specify the name of a parameter. Add this parameter to the P8 or Pro Panel discipline components. The value of the parameter in the discipline components defines the variant of a P8 window macro or a Pro Panel macro.

Valid values for P8 lie in the range of A to Z.

Valid values for Pro Panel are any strings. Default is 1.
