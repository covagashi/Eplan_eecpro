## Parameter Name For Part Attributes

Path to model variable:

Disciplines > ECAD > Eplan Electric P8 > Eplan Data Zip (.edz) generation

The model variable Parameter name for macro path specifies the name of the parameter, which stores the path to the P8 macros.

Default value is MacroName.
