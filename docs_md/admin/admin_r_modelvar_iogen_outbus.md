## Parameter Name For The Outgoing Bus Line

Path to model variable:

Disciplines > IO Generator

Specify the name of a parameter. Add this parameter to the PLC discipline components. The string value of the parameter in the discipline components defines the name of the outgoing bus. For the connection of the incoming bus, the [Parameter Name For The Incoming Bus Line](admin_r_modelvar_iogen_inbus.htm) model variable has to be specified.
