You are here: 

## Tables

Path to the preferences:

Menu Window > Preferences

Dialog Preferences > General > Tables

With the options for automatic setting of the column widths you can define whether the columns of tables are to be set automatically, only during opening or only when updating.

Instead of the automated customizations you can specify a fixed width.

Optionally you can calculate only the result column on request by pressing the key [F6] or [F7].
