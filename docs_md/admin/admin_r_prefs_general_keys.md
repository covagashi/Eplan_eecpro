# Keys

This functionality is only available for certain module packages. [Info / Copyright](license.htm)

You are here: 

## Keys

Path to the preferences:

Menu Window > Preferences

Dialog Preferences > General > Keys

The preferences for keys show the current affiliation of commands to shortcut keys. The shortcut keys can be customized in any way. The table can both be filtered as well as be sorted by columns.

The current affiliation can be exported into a file in CSV format.

Conflicts due to a duplicate affiliation are sorted by commands and displayed in the window Conflicts.
