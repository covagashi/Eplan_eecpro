# Colors and fonts

This functionality is only available for certain module packages. [Info / Copyright](license.htm)

You are here: 

## Colors and fonts

Path to the preferences:

Menu Window > Preferences

Dialog Preferences > General > Representation > Colors and fonts

Under Library Project Editors Theme the colors for elements of the Parameter editor can be determined for library and project components. This way it is possible to adjust the background colors of marked, editable and non-editable fields, as well as the font color. The background color for fields with validation errors and warnings can furthermore be determined.

Under Project Catalog > Color for inactive objects deactivated objects in the project catalog can be changed and highlighted with regard to colors or typography. In this way inactive objects can also be detected without activating the filter.
