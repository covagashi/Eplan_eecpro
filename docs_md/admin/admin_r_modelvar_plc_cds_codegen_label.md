# Comment for generated project labels

This functionality is only available for certain module packages. [Info / Copyright](license.htm)

You are here: 

## Comment for generated project labels

Path to model variable:

Disciplines > PLC > CoDeSys > Code Generator

Specify a string value. The value determines the text that is attached as a comment to the created project labels.
