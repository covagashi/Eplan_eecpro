# Exporting the current information model

This functionality is only available for certain module packages. [Info / Copyright](license.htm)

You are here: 

## Exporting the current information model

The entire information model is exported to an XML file, and subsequently imported into an empty database of the new installation (note: The XML format is database-neutral and allows for the switch between different database products and versions).

The entire information model is exported to an EOX file, and subsequently imported into an empty database of the new installation.

  1. Click File > Export....

The Export dialog opens.

  2. Navigate to Model > Model data (EOX).
  3. Click [Next >].

The EOX export dialog opens.

  4. Enter a file name and storage location for the export file.
  5. Use the Libraries option to select all libraries and the Projects option to select the complete model, or subsequently deselect entries that should not be exported.
  6. Click [Finish].
