## Parameter name for project user supplementary fields

Path to model variable:

Disciplines > ECAD > P8

Specify the name of a parameter. Add this parameter to the ECAD discipline components. The value of the parameter in the discipline components defines the name and the value of the project-specific user supplementary field. The assignment takes place as a key-value pair.
