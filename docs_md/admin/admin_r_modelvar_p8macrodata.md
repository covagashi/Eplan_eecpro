## Parameter Name For The Macro Parameter Structure

Path to model variable:

Disciplines > ECAD > Eplan Electric P8

Specify the name of a parameter. If the value of the model variables Write macro-parameter structure in parameter is set to true, the parameter with the specified name is created when a P8 macro is read in. All the relevant data are read from the P8 macro and stored as a list of maps in the parameter. The list contains, among other things, information about variants, text markers (plugs and sockets), variables, value sets, representation type, and others.
