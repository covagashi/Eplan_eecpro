## Parameter name for the location

Path to model variable:

Disciplines > IO Generator

Specify the name of a parameter. Add this parameter to the PLC discipline components. The value of the parameter in the discipline components defines the location of the discipline components within the configured machine or plant. Through the location you can, for example, group the associated components.
