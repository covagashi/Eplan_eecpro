# Default Microsoft Office Version

This functionality is only available for certain module packages. [Info / Copyright](license.htm)

You are here: 

## Default Microsoft Office Version

Path to model variable:

Model > Model Variables > Datasources > (Obsolete) Default Microsoft Office Version

Specifies the version of Microsoft Office that is used as the default for new MS Access and Excel data sources.

Microsoft Access is no longer supported for data sources.
