## EEC Job Server Add-On

The optional function EEC Job Server Add-On provides the functions to extend a stand-alone installation with Job Server capabilities.

To install the optional function:

  1. Start EEC
  2. Open Help > Install new software....
  3. In the Use field select the \--All available sites-- or de.eplan.eec.install.optfeatures entry from the drop-down list.
  4. Expand the EEC Optional Features path.
  5. Select the EEC Job Server Add-on entry.
  6. Click [Next >].



The function to be installed is only listed so that the selection can be checked.

  1. Click [Next >].
  2. Accept the license terms.
  3. Click [Finish].
