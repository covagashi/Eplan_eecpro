## Parameter Name For The DB Number

Path to model variable:

Disciplines > PLC > STEP 7 > SIMATIC

Specify the name of a parameter for this model variable. Add this parameter to the SIMATIC discipline components. The value of the parameter in the discipline components defines the number of data blocks.
