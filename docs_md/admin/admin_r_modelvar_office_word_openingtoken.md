## Opening Token For Parameter Replace Sequence

Path to model variable:

Disciplines > Office > Word

Specify a value. This value defines the opening token by which the parser detects the beginning of a parameter sequence specification. To do this use a combination of numbers that usually cannot occur in the text of a Word document. During the generation of a Word document a parameter sequence is replaced by the value of the parameter.
