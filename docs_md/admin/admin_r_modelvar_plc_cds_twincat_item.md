## Parameter Name For The Item Shortcut

Path to model variable:

Disciplines > PLC > CoDeSys > TwinCat

Specify the name of a parameter for this model variable. Add this parameter to the TwinCat discipline components. The value of the parameter in the discipline components defines the Tree Item Shortcut.

Valid values are TIIC, TIID, TIRC, TIRT, TIRS, TIPC, TINC, TICC, TIAC.
