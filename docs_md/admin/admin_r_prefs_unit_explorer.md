# Preferences for Unit Explorer

This functionality is only available for certain module packages. [Info / Copyright](license.htm)

You are here: 

## Preferences for Unit Explorer

Path to the preferences:

Menu Window > Preferences

Dialog Preferences > Unit Explorer

The structure element Unit is specified by the following preferences:

     * Units in alphabetical order sorts the units in the library catalog alphabetically.
     * Objects in alphabetical order sorts the objects in the library catalog alphabetically.
     * Show filter conflict dialog creates an error message, if an enabled filter setting for the library catalog avoids that particular component types cannot be displayed for synchronization.
