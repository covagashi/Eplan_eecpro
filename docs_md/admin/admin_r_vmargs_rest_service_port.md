## de.eplan.eec.rest.service.port

The following EEC argument must be transferred with the following syntax:
    
        -D<Argument>=<Value>

EEC argument |  Usage  
---|---  
-Dde.eplan.eec.rest.service.port=<Port number> |  Only for eeLide  
**Annotation**  
A port address must be specified for the connection with the REST service via the browser Example:
    
        -Dde.eplan.eec.rest.service.port=9191
