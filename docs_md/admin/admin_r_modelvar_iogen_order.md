## Parameter name for the sort order

Path to model variable:

Disciplines > IO Generator

Specify the name of a parameter. Add this parameter to the PLC discipline components. The value of the parameter on the discipline components defines the order in which the addresses for components are assigned (see [Order of actuators and sensors](eecplc_k_modeling_order_of_sensors_actuators.htm)).
