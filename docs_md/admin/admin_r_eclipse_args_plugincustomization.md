## -pluginCustomization

Eclipse argument |  Values |  Usage  
---|---|---  
-pluginCustomization |  propertiesFile |  optional  
**Annotation**  
By default, configuration data of plug-ins are specified via the file config.ini. If the argument -application specifies an alternative application, the plug-ins will generally have to be adapted specifically as well. This is done via the argument -pluginCustomization.
