## Parameter Name For The Macro Path

Path to model variable:

Disciplines > ECAD > Eplan Electric P8 > Eplan Data Zip (.edz) generation

The model variable Parameter Name For The Macro Path specifies the name of the parameter, which stores the path to the P8 macros.

Default value is MacroName.
