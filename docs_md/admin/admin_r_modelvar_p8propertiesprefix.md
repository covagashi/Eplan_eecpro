This functionality is only available for certain module packages. [Info / Copyright](license.htm)

## Prefix For Eplan Electric P8 Properties

Path to model variable:

Disciplines > ECAD > Eplan Electric P8

Specify a value. This value defines the prefix that precedes the parameter name so that Eplan Electric P8 interprets the value of this parameter as a property.

The default is PropId_.

For example the company name is specified with a parameter of the PropId_10015 name.
