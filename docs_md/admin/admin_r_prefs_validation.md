# Preferences for validation

This functionality is only available for certain module packages. [Info / Copyright](license.htm)

You are here: 

## Preferences for validation

Path to the preferences:

Menu Window > Preferences

Dialog Preferences > Formulas > Validation

If the Observe conditions when checking formula results option is selected, the entered conditions for the parameters are checked when the formulas are evaluated. Without this option the conditions are ignored. By default this option is activated.
