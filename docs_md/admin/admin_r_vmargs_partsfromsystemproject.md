# PartsFromSystemToProject

This functionality is only available for certain module packages. [Info / Copyright](license.htm)

You are here: 

## PartsFromSystemToProject

The EEC argument must be transferred with the following syntax:
    
        -D<EEC-Argument>

EEC argument |  Usage  
---|---  
-DPartsFromSystemToProject |  Optional  
**Annotation**  
Performs a part synchronization with the Eplan parts database. By passing a part number, the terminal descriptors and the technical specifications are synchronized with the parts database. Example:
    
        -DPartsFromSystemToProject
