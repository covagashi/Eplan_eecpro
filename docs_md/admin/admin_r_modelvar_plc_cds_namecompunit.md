This functionality is only available for certain module packages. [Info / Copyright](license.htm)

## Name of the unit for the CoDeSys components

Path to model variable:

Disciplines > PLC > CoDeSys

Specify a string value. The value determines the unit in which a search for parameters is carried out while discipline resources are added and synchronized before a new parameter is created in the unit.
