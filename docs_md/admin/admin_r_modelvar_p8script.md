## Parameter Name For C# Script

Path to model variable:

Disciplines > ECAD > Eplan Electric P8

Specify the name of a parameter. Add this parameter to the P8 discipline components of the WiringDiagram type. The value of the parameter in the discipline components defines the absolute name of a C# script. The path to the C# script has to be specified with double backslashes, for example =c:\\\scripts\\\ecad\\\myP8Script.cs.
