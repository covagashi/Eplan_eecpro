This functionality is only available for certain module packages. [Info / Copyright](license.htm)

You are here: 

## Calling the Web EEC

The Web EEC is called in a browser with the following syntax:
    
        http://<Hostname>:<Portnummer>/webec/main

## [Example](javascript:void\(0\);)

If, for the Web EEC, as described before, it was installed in the folder webapps\webec of a Tomcat server that accepts http requests at http://localhost:8181:

<http://localhost:8181/webec/main>
