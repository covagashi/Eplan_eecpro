## Exporting user preferences

Upon installation of a new EEC version, the default user preferences are loaded. To apply the familiar user preferences to the new installation, you have to export them first. The user preferences are stored in a compressed format.

This is how you export the user preferences:

  1. Click File > Export....

The Export dialog opens.

  2. Navigate to General > Settings.
  3. Click Next >.
  4. Select the Export all option.
  5. Enter a file name and storage location.
  6. Click Finish.



Result:

The user preferences are stored in an .EPF file.
