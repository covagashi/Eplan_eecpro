## Parameter Name For Updating Reports

Path to model variable:

Disciplines > ECAD > Eplan Electric P8

Specify the name of a parameter. Add this parameter to the P8 discipline components of the WiringDiagram type. The value of the parameter in the discipline components controls the update of the reports.

Valid parameter values are true and false.
