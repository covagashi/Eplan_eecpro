## Application

Path to the preferences:

Menu Window > Preferences

Dialog Preferences > General > Application

     * The Show used backend in window title option activates the display of the absolute path to the back-end in the header of the EEC window.

The option is active as a default.

     * The option Show application exit confirmation dialog activates the overlay of a dialog when an EEC is completed.

The option is active as a default.
