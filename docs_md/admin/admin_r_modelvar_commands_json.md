## Parameter name for the JSON structure

Path to model variable:

Model > Model variables > Commands > SetParameterValueFromJsonCommand

Specify the name of a parameter. Add this parameter to the discipline components. The value of the parameter in the discipline components is assigned by the Job Server with the key-value pairs of a JSON structure.
