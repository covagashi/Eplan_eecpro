## \--launcher.ini

Eclipse argument | Values | Usage  
---|---|---  
\--launcher.ini | Name of the ini file | Optional  
**Annotation**  
This argument is used to reference a different initialization file. Example: ec.exe --launcher.ini user1.ini points to the initialization file user1.ini. Without this argument, EEC will use the initialization file of the same name. Examples: ec.exe uses ec.ini. ec_admin.exe uses ec_admin.ini.
