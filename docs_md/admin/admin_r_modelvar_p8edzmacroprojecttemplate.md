This functionality is only available for certain module packages. [Info / Copyright](license.htm)

You are here: 

## Macro project template

Path to model variable:

Disciplines > ECAD > Eplan Electric P8 > Eplan Data Zip (.edz) generation

Specifies the name of the macro project template. The macro project template is only required for the EDZ+ generation. Since a macro project template hosted by Eplan is used, the specification of a customer-specific macro project template is not possible.

Default is IEC_bas001.zw9.
