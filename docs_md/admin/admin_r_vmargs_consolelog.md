## consolelog

EEC argument |  Usage  
---|---  
-consolelog |  Optional  
**Annotation**  
Opens at the launch a console window and logs in it the start-up process. Alternatively, -consolelog can also be appended as a starting parameter to the call of the executable file. Example:
    
        -consolelog
