## Closing token for REPLACE variables

Path to model variable:

Disciplines > PLC

Specify a value. This value defines the closing token by which the parser detects the end of a Replace variable specification. To do this use a combination of numbers that usually cannot occur in the PLC code. The Replace variable is replaced during the generation of the PLC code.
