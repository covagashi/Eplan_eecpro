## Parameter Name For The Window Macro Representation Type

Path to model variable:

Disciplines > ECAD > Eplan Electric P8

Specify the name of a parameter. Add this parameter to the P8 or Pro Panel discipline components. The value of the parameter in the discipline components defines the representation type of the window macro.

Valid values are:

     * MultiLine
     * SingleLine
     * PairCrossReference
     * Overview
     * Graphic
     * Panel layout
     * PI_FlowChart
     * MultiLine_Fluid
     * Topology
     * Functional
     * Preplanning
     * FunctionOverview_Fluid
