# Optional functions

This functionality is only available for certain module packages. [Info / Copyright](license.htm)

## Optional functions

The installation can be extended with optional functions.

Currently available optional functions:

Name | Function  
---|---  
[EEC Diagnostics Add-On](admin_h_installation_optionalfeature_diagnostics.htm) | Provides the diagnostics functions for EEC  
[EEC Job Server Add-On](admin_h_installation_optionalfeature_jobserver.htm) | Provides the Job Server functions for EEC  
[EEC Job Server Worker Add-On](admin_h_installation_optionalfeature_jobserverworker.htm) | Provides the Job Server Worker functions for EEC  
[EEC P8 Integration Service Add-On](admin_h_installation_optionalfeature_p8_integration.htm) | Provides a function which makes it possible to open the corresponding discipline component in EEC from a P8_Macro.  
[EEC SAP RFC Add-On](admin_h_installation_optionalfeature_sap.htm) | Provides functions for SAP RFC support using JCo for EEC.
