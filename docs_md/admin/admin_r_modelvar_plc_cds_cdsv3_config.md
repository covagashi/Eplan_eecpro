This functionality is only available for certain module packages. [Info / Copyright](license.htm)

## Name of CoDeSys V3 configuration file

Path to model variable:

Disciplines > PLC > CoDeSys > CoDeSys V3

Specify a string value. The value determines the name of a CoDeSys V3 configuration file. The specified file is written during the generation of the PLC code.
