## Notes on the scope of performance

This documentation describes the complete scope of performance of the software. This scope is available to you if you have licensed all products and modules. If this is not the case, the documentation goes beyond the scope of the purchased software, this means it also describes functions that are not available to you.

Relevant for the scope of the software purchased by you is solely the performance description of the software licensed by you. For any scope exceeding this, this documentation therefore only functions as information on the scope which you can additionally purchase.
