# OrderedMap methods

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  OrderedMap methods OrderedMap methods The OrderedMap offers all methods of Map. Methods of Map with Map as the return value, return a OrderedMap for OrderedMaps. The method collect and keys return lists for OrderedMaps, because the keys of OrderedMaps have an order.
