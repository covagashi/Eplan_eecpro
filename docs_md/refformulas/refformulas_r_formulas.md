# Formulas

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Formulas Formulas can be used to calculate parameter values of project components and in disablers. The syntax of the formulas is based on the Object Constraint Language (OCL) which is defined in the Unified Modeling Language Specification (UML) Version 1.5. For the use in the functional configuration the syntax of the OCL is slightly adapted. The following describes all usable formulas for each type of object.
