# documentRef(String absoluteName)

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  documentRef(String absoluteName) Navigation methods for project components Checks if the document (e.g. a diagram) given by absoluteName is referencable. If the superordinate component, which contains the document, is disabled, no document can be referenced. **documentRef(String absoluteName)**  
---  
Arguments | String | absoluteName | The absolute path to the document  
Return value | Boolean |
