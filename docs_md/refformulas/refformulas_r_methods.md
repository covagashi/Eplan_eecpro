# Methods

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Methods For each object of the object model executable methods are defined. One method of the String objects for example is the calculation of the length of the character string. Methods are defined by their signature. The signature of the method is made up of the name of the method and all its arguments. For example the method size() for Strings has the name size and no arguments. If methods do not have any argument, they can also be called without parentheses!
