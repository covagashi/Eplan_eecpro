# image(Object keyEO)

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  image(Object keyEO) This method returns an image for an object via an image register. For this method the keyEO is also the contextEO at the same time. **image(Object keyEO)**  
---  
Argument | Object | keyEO | Image register or image register extension that contains the image object  
Return value | image |
