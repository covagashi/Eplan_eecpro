# ifNull(Block expression)

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  ifNull(Block expression) The condition ifNull returns the block expression, if the receiving object is <null>, otherwise the receiving object is returned as the result. **ifNull(Block expression)**  
---  
Argument | Block | expression | Expression to be returned if the object is <null>  
Return value | Result or block expression

---|---
