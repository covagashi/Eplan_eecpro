# Data protection

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Data protection The Quality Feedback Report is a ZIP file and essentially contains the configuration details, settings and descriptions of current or previous errors and events. Except for the model, all files contained in the ZIP file are text files. Before sending the Quality Feedback Report, you can unzip the ZIP file to delete in the text files whatever you do not want to send.
