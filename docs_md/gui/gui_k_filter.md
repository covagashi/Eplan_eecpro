# Filter

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Filter To improve clarity in case of larger models, it helps to filter certain objects from a variety of objects and hide the ones that are not needed. For this purpose, filters can be set up for views. The Filter dialog is opened via the  symbol of the view-specific toolbar. Filters are always subtractive. The filter is switched on or switched off via the symbol  of the view-specific toolbar. The view is updated after the settings have been confirmed.
