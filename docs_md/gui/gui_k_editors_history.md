# Editor history

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Editor history The editor history allows the user to open previously edited editors again. Using the two icons , , the user can navigate to the previous and subsequent editors. The icons are located on the global toolbar. To switch to a specific editor in the history, a list of editors can be opened using the small triangles next to the icons.
