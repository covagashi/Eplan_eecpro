# Synchronization of views and editors

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Synchronization of views and editors Each editor is logically linked to an object within a view. When switching between two editors, it often helps to also select the relevant object in opened views. If the object being edited in the editor is to be selected in the view, synchronization between the editor and the view is possible. To do this, use the  button in the view. This turns on synchronization. When switching between editors, the corresponding element for each editor is displayed in the synchronized view. Synchronization can be enabled and disabled via the  button.
