# Starting EEC

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Starting EEC EEC is started via the Start menu using the Start EEC icon and / or by executing the ec.exe file. Depending on the configuration the program start is carried out in different sequences. No explicit model specification is configured:
    1. Query of the data management (EOX file).
    2. Display of the EEC user interface with the Engineering perspective.
A model specification is configured and EEC is started for the first time:
     * Display of the EEC user interface with the Engineering perspective.
A model specification is configured and EEC is started again:
     * Display of the EEC user interface with the perspective last used.
The following information can be found in the title bar of the EEC user interface:
     * EEC version.
     * If required, a restriction of the access rights to the data management.
     * The data management used.
