# Internal structure of editors

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Internal structure of editors The following figure shows the internal structure of an editor. It consists of the areas Editor menu (A) and Editor page (B):
    1. The editor menu contains different menu items that open an editor page.
    2. If a further editor is selected, it is displayed directly below the existing editor.
    3. By default only one editor page is visible. However, several editors can be opened by pinning individual editors by using the icon .
