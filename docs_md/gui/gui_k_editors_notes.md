# Notes

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Notes The editor for notes can be used to assign notes that are available in the text (*.txt) or Word (*.docx) format to objects. In the notes editor, the icon Add opens a dialog for selecting a note resource. The search path for notes can be configured in the settings under Disciplines.
    1. The selected resource is listed in the table of the editor.
    2. The content is displayed below the table in an editor window for the respective format, where it can also be edited.
Note If files are to be used for notes that the system will not recognize as being in the text format, you can add these to the content types in the text format under Window > Settings > General > Content types > Text.
