# Hyperlink navigation

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Hyperlink navigation In EEC, it is common for information to be displayed in views and editors that is defined elsewhere. The following screenshot shows, for example, a formula that is displayed in the editor of a project component, but was defined in a library component. Using the hyperlink navigation, it is possible to switch from project objects to referenced objects. To do this, press the ALT key on the orange-color field of the editor, so that a hand appears. While keeping the [Alt] key pressed, left-click the button, and the view switches to the editor of the library object. The following figures show how the mouse pointer becomes a hand when the [Alt] key is pressed on the orange-color field. A left click establishes the switch to the library object.
