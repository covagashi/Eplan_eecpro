# Call

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Call The Quality Feedback wizard appears automatically with an exception error if the overall system identifies an error in a program component for which no error handling exists yet. Before the program is terminated automatically after an exception error, in order to prevent loss of data, the Quality Feedback wizard appears. For program errors that do not cause the program to be terminated, call the Quality Feedback wizard manually:
    1. Select the menu item File > Export....
    2. In the Export dialog, select in the folder Other the entry Quality Feedback.
    3. Click [Next >].
