# Create and send

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Create and send In the upper text field, enter the steps required to reproduce the error. For example, describe which check boxes, menu items and buttons must be activated, and in which order, in order to arrive at the error. If you have called the Quality Feedback wizard manually, please explain how to recognize the error and what should be the correct way. The Quality Feedback Report can be prioritized using three levels of priority. The three levels are Low, Medium, and High. Without a selection, the default priority is Low. The priority is also displayed in the file name of the Quality Feedback Report. Example: Report_low_2008-08-14_18-23.zip In addition, the priority is also stored in the file report-details.txt in the ZIP file. To modify the preferences for file name and folder in which the Quality Feedback Report is to be created, click [Browse...] and select a new target file. If the error can be reproduced only with your data, activate the check box Add model. Please note that this could increase the file size substantially, and delay the creation. When the Quality Feedback wizard is called manually, create the file by clicking [Finish]. Please send the file as an e-mail attachment to [support@eplan.de](mailto:support@eplan.de). If the Quality Feedback wizard has opened automatically, you can create the file by clicking [Save]. By clicking [Send by e-mail], the default e-mail program of your operating system will open as well. Add the file created to the e-mail as an attachment manually, and click [Send].
