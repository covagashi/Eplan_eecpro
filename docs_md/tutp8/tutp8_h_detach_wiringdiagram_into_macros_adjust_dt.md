# Set device tag

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Set device tag Insert a device tag to identify the sensors:
    1. Mark all the sensors on the 1b Sensors page.
    2. Select the following commands: Tab Home > Command group Edit > Properties.
    3. The Properties (components): General devices opens.
    4. If required, open the Function tab.
    5. In the Displayed DT field, enter the text <DT>.
    6. Confirm with [OK].
    7. The message window Settings: DT syntax check opens.
    8. Click [Ignore].
    9. The message window is displayed for each sensor and has to be closed correspondingly often by using [Ignore].
