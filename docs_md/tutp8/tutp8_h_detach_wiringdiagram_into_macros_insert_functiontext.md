# Insert function texts in page

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Insert function texts in page The function texts of the PLC inputs are to be inserted by the EEC while creating. Therefore, the existing texts are replaced by function texts:
    1. Double-click the first PLC input I0.0.
    2. The Properties (component) PLC and bus ports window opens.
    3. In the Function text field, exchange the text with the following:
    
        <Functiontext1>

    1. Confirm with [OK].
    2. Repeat steps 1 to 4 for each further PLC input, and use ascending numbers for the function texts (<Functiontext2>, <Functiontext3>, etc.).
