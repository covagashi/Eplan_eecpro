# Insert schematic and page into mechatronic components

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Insert schematic and page into mechatronic components In the first step, discipline components for the schematic and page must be linked with the mechatronic component Feeder:
    1. Open the Feeder component.
    2. Using Drag & Drop, drag the Schematic component once, and the Page component twice into the component panel of the editor.
    3. Name Page and Page2 to PLC_Sensors_1_8 and PLC_Sensors_9_16.
