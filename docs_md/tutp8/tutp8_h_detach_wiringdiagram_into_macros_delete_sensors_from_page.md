# Delete sensors from page

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Delete sensors from page The window macro of the page should not contain any sensors. They must be removed from the page:
    1. Double-click the 1a Page page to open it.
    2. Highlight all sensors.
    3. Delete the sensors using the [Delete] key.
