[Info / Copyright](license.htm) You are here:  Divide Eplan Electric P8 schematic into macros The loaded macro project is to be used to create macros. First, copy the 1 Sensors page twice. Using one copy, create a window macro of a page in which all sensors have been deleted (see the following figure). The other page is not required for the window macro. Using the second copy, a fragment is created of each unique sensor. First, though, all duplicate sensors as well as the PLC inputs and power supply lines must be deleted. Information: The copies ensure that the macros can be assigned to the corresponding original page. This assignment must be maintained, so that subsequent changes to the macro project can be synchronized with EEC.
