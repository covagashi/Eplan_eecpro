# Generating the current status of the schematic

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Generating the current status of the schematic Generate the current status of the schematic.
    1. In the Project Explorer select the Mechatronic component.
    2. In the shortcut menu, select ECAD > Generate_structure.
    3. In parallel to Mechatronic, a structure consisting of the discipline root node ECAD, a schematic (Schematic_Feeder), and two pages (PLC_Sensors_1_8_Feeder and PLC_Sensors_9_16_Feeder) has been created.
