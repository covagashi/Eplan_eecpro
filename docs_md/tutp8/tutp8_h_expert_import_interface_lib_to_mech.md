# Importing the library for interfaces into the modular system

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Importing the library for interfaces into the modular system To be able to use the interfaces in the modular system, the T_Interfaces library has to be imported to the mechatronic modular system.
    1. Open the T_Mechatronic_ModularSystem library.
    2. Change to the Imported libraries editor page.
    3. Click  to open the Finder dialog.
    4. Click [Search] in the Finder dialog.
    5. Select the T_Interfaces library.
    6. Confirm with [OK].
    7. Save the T_Mechatronic_ModularSystem library.
    8. The interfaces of T_Interfaces can now be applied in the T_Mechatronic_ModularSystem.
