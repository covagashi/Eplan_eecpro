# Generating all macros

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Generating all macros All preparations for the macros are completed. Now all macros will be generated together:
    1. Select the following commands: Tab Master data > Command group Macros > Generate automatically.
    2. The dialog Generate macros automatically opens.
    3. Mark the Overwrite existing macros option.
    4. Mark the Apply to entire project option.
    5. Confirm with [OK].
    6. The dialog Generate macros automatically is closed.
    7. Exit Eplan Electric P8.
The macros will be stored in the following directories:
    
        <EEC installation folder>\resources\ECAD\Tutorial\P8\Page
    <EEC installation folder>\resources\ECAD\Tutorial\P8\PLC_Inputs
