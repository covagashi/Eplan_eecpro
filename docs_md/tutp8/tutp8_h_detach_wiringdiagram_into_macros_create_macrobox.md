# Create macro box

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Create macro box Create a macro box that combines all symbols of this page into a macro. Note Make sure that the anchor of the placeholder object is inside the macro box; otherwise, the macro cannot be created correctly.
    1. Select the following commands: Tab Insert > Command group Navigator > Insert macro box.
    2. Drag a frame around the power supply lines and PLC inputs.
    1. End the Insert macro box function using the [ESC] key.
Define the properties of the macro box.
    1. Highlight the macro box.
    2. Select the following commands: Tab Home > Command group Edit > Properties.
    3. The Properties (components): Macro box opens.
    4. In the Name field, enter the file name Page\Page.
