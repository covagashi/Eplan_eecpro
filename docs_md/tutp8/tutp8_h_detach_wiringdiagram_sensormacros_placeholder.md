# Create placeholder objects for sensors

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Create placeholder objects for sensors Analog to the schematic page a placeholder object has to be added for each sensor to equip the macros with variables.
    1. Select the following commands: Tab Insert > Command group Macros > Navigator > Insert placeholder object.
    2. Open a frame for each sensor that includes all the elements of the sensor.
    3. The dialog Properties (components): Placeholder object opens.
    4. Open the popup menu of the table.
    5. Select Adopt variables.
    6. The Variable column is then filled with the name of the variables.
Note The variables for the socket and plug parameters must not be entered in the Variable column, so that no duplicate objects can be created during the automatic application of the parameters.
    1. From the Variable column remove the variable entry for the plug parameter (#<Plug:IP>).
    2. Open the Values tab.
    3. From the Variable column remove the variable entry for the plug parameter (#<Plug:IP>).
    4. Confirm with [OK].
