## Graphical editor

Diagrams of the type specified in the configuration (planning diagram or Gantt chart) are created using the graphical editor. Once a diagram has been inserted into a project, the graphical editor appears with the editing area empty.

The graphical editor for planning diagrams consists of the following elements (see the following diagram):

1\. Toolbar,

2\. Palette, and

3\. Diagram area.
