## Deleting nodes

Nodes can be deleted in a way that either affects only the diagram and the project components represented in it or that also deletes the project components themselves.

When deleting project components that have connections to parents and/or children, care should also be taken to delete the references to these components in the corresponding parameters (see [Deleting references to nodes](eecgraph2d_k_edges_delete_references_to_node.htm)).
