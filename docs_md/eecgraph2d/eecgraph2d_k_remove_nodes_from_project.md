This functionality is only available for certain module packages. [Info / Copyright](license.htm)

## Deleting nodes from the project structure

Project components represented in the diagram may be deleted from the project structure together with nodes represented in the diagram by selecting Delete from model from the context menu.

This only causes references to parent and/or child components in the corresponding parameters to be deleted when permitted by the diagram definition (see [Deleting references to nodes](eecgraph2d_k_edges_delete_references_to_node.htm)).
