## Positioning slave nodes on the edge of master nodes

If no position attribute has been assigned for slave nodes, all slave nodes that have the same value for the direction attribute are distributed evenly along the length of the edge of the master node. The position of a slave node on the edge of a master node can be specified via the position attribute either as a percentage or in pixels. As the value increases, the position of the slave node changes in a clockwise direction along the edge of the master node (see the below diagram).
