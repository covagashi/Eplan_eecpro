## Decorating nodes

Node decoration can be configured at a superordinate level and overwritten by each specific node definition.

Decorators are as follows:

     * Border color when unselected (boColor),
     * Border color when selected (selectedBoColor),
     * Border width (boWidth),
     * Distance from content to border (boMargin),
     * Border line style (boLineStyle),
     * Background color when unselected (bColor),
     * Background color when selected (selectedBColor),
     * Border color of an automatically inserted slave node (autoSlaveBoColor),
     * Border color of slave nodes when the corresponding master node is selected (highlightBoColor),
     * Font color when unselected (fColor),
     * Font color when selected (selectedBColor),
     * Predefined width (prefWidth),
     * Predefined height (prefHeight),
     * Transparency (transparency)
