## Deleting graphical objects and components from the diagram

Deleting a graphical object from the diagram with the [Del] key or via Context menu > Delete from the diagram does not affect the model; it only deletes the graphical object. However, if a graphical object in a diagram is deleted by pressing [Shift]+[Del] or via Context menu > Delete from the model, the project component will also be deleted from the model.This functionality is only available for certain module packages. ## Generate connections successively

With [Shift] pressed, it is possible to insert connections successively. When [Shift] is released, this function ends once the last connection has been completed.
