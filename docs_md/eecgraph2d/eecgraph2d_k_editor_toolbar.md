## Toolbar

The toolbar offers the following functions:

     * Undo: Undoes the last change.
     * Repeat: Repeats the last change.
     * Zoom factor: Sets the editing area to the selected zoom factor.
     * Background image: Opens a dialog for selecting a background image for the editing area.
     * Alignment: Aligns the selected nodes. A selection can be opened via the arrow to the right of the icon.

The selection icons (see the following illustration) can be used to align the selected nodes with one another or with the diagram-editing area.
