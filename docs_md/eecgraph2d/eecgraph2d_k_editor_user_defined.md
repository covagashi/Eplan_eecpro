## User-defined graphic editors

A diagram configuration is created as an XML file in the modular-system library. It determines the appearance of the diagram and the behavior of nodes and links. A guide to the individual XML elements used in configuring diagrams may be found in the [GRAPH2D](refgraph2d_r_graph2d.htm) reference section.

Multiple diagram configurations can be created, e.g. to make it possible to create both a planning diagram and a Gantt chart.

The following diagram presents a schematic of an information model whose library contains a diagram configuration. An excerpt from the diagram configuration reveals its structure. In the project, a diagram used in the graphical configuration can be inserted at a predefined position in the model. The diagram can represent all or part of the project.
