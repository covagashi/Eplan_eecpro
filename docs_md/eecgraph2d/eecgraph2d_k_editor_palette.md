## Palette

The palette appears at the right edge of the diagram editor, but can be docked at any position.

It primarily contains selection tools (1.). When planning diagrams are being edited, the palette also contains notational tools (2.).

In addition, groups (3.), components for instantiation in the diagram (4.), and connections (5.) can be listed in the diagram definition (see the following illustration).

The context menu (see the following illustration) can be used to select how palette elements should be displayed.

Possible settings are:

     * Layout > Columns displays one icon per column with text (see the above diagram).
     * Layout > List standard view, displays icons and texts.
     * Layout > Icons Only displays only icons.
     * Layout > Details displays icons, texts and tooltips.
     * Use large icons Both a small icon (smallIcon attribute) and a large icon (largeIconattribute) can be defined for each palette entry in the diagram configuration. Which icon is displayed depends on the setting.
