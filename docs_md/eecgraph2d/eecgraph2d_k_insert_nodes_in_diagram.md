## Inserting nodes into a diagram

Nodes can be placed in a diagram both manually and automatically. Automatic positioning is limited here to nodes from components that are already available in the project structure.

Nodes from existing project components can be positioned manually. However, by means of an action, project components can also be instantiated immediately before the node is positioned.
