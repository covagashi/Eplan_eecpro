## Insert diagram in project

In the diagram configuration, the user can define which position in the project a diagram can be inserted at. A diagram can be inserted in two ways at the defined position.

     * By dragging and dropping (instantiating) the diagram definition into the project (see the following screenshot).

     * By selecting Insert > Diagram name from the context menu of the component.

For the Bodyshell Work project, two diagram types (Planning_Diagram and Procedure_Diagram) have been configured, so that both diagrams can be inserted for a single Plant.
