# Deleting graphical objects and components from the diagram

This functionality is only available for certain module packages. [Info / Copyright](license.htm)

You are here: 

## Deleting graphical objects and components from the diagram

Deleting a graphical object from the diagram with the [Del] key or via Context menu > Delete from the diagram does not affect the model; it only deletes the graphical object. However, if a graphical object in a diagram is deleted by pressing [Shift]+[Del] or via Context menu > Delete from the model, the project component will also be deleted from the model.
