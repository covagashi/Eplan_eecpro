## Zoom for the timescale of a Gantt chart

The timescale of a Gantt chart can be changed by zooming in and out (see the following screenshot). By selecting Time Zoom > Zoom In or Zoom Out in the context menu, the timescale can be stretched or compressed. Time Zoom > No zoom can be used to reset the zoom factor of the timescale back to value defined in the configuration file.
