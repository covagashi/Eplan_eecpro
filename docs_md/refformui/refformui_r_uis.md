# uis

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  uis The Form-UI element <uis> is the top-level element. All following Form-UI elements are wrapped by this element. The element does not have any attributes. Allowed sub-elements | Quantity  
---|---  
[form](refformui_r_form.htm) | any  
[Example code](javascript:void\(0\);)
    
        <uis>
    	<form title="Formular 1" id="fm1" >
    		<label>Formular 1</label>
    	</form>
    	<form title="Formular 2" id="fm2" >
    		<label>Formular 2</label>
    	</form>
    </uis>

In order a Form-UI is created with two empty Form-UIs: Note In the following sections most of the code examples are shown without the enclosing elements <uis> and <form>, in order to set the relevant code to the foreground If these code examples are to be used for own Form-UIs, the elements <uis> and <form> have to be added.
