# XML schema for Form-UI

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  XML schema for Form-UI The demo model Docu_UI_Configuration contains the elements which are described in the following. These can be easily transferred to an own project by copy and paste. For the programming of user interfaces, the EEC provides an XML editor (see [XML editor for Form-UIs and diagram definitions](eecbase_k_editors_xml_editor_for_formui_and_diagramdefinitions.htm)): The Form-UIs can also be edited with Notepad from Windows or any editors of third-party distributors, e.g. [XML-Spy](http://www.altova.com), [Liquid XML-Studio](http://www.liquid-technologies.com), [Oxygen XML-Editor](http://www.oxygenxml.com), etc. Programmers of user interfaces which use third-party editors are provided with a XML-schema for validation. The schema file forms.xsd is provided in the folder ..\install\forms.
