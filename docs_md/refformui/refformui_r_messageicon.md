# messageIcon

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  messageIcon The <messageIcon> element defines the type of message window by assigning a specific icon. Attribute name | Usage | Attribute values | Default value | Description  
---|---|---|---|---  
type | required | Question,  
Information,  
Warning,  
Error |  | Displays an icon in the message window.  
Question = Question mark in a blue circle  
Information = Exclamation mark in a blue circle  
Warning = Exclamation mark in a yellow triangle  
Error = X in a red circle  
Allowed sub-elements | Quantity  
---|---  
none |
