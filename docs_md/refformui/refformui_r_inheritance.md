# Inheritance

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Inheritance Form UIs that are defined in a Class A (for example, the pages structure and field types in the documentation) are inherited in the subclass B. In the subclass B further specific forms for B (e.g. the page subclass in the documentation) can be specified. Form-UIs of the parent class can also be alternatively overwritten by the subclass.
