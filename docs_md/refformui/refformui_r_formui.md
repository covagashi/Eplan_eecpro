# Form-UI

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Form-UI The Form-UI is a XML-defined user interface that is used to define user-specific views on the mechatronic model. The user interface is displayed in EEC independent of the UI technology and by means of appropriate adapters displayed in a web browser, a swing client or a rich client or directly in EEC.
