# Navigating from a Form-UI element to the definition in the configuration file

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Navigating from a Form-UI element to the definition in the configuration file The modular system developer can access the definition of an element in the configuration file directly from the Form-UI. In the Form-UI window, an arrow is first displayed on each Form-UI element with the key combination [Ctrl] \+ [Alt]. The arrows are hidden by pressing the key combination again. By clicking on an arrow, the XML file of the Form-UI is opened in a new Editor and the cursor is placed on the relevant element. Note This function is currently only available for displaying Form-UIs with SWT.
