# Tooltips

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Tooltips By means of tooltips, information can be displayed for certain elements, when the user moves the mouse cursor over the element. Tooltips are specified as an attribute (see Tooltip as Attribute) or as an element (see tooltip). If a Form-UI in a popup window is displayed as a tooltip, all elements of Form-UIs are usable, even inputs are possible. The entered of selected values are only accepted if the popup window is **not** closed with the ESC key.
