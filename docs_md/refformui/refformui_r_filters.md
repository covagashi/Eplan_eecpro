# filters

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  filters The <filters> element activates the filters for displaying the determined values of the created <pureTable> table. The <filters> element encloses the filters, which are defined by the [columnFilter](refformui_r_columnfilter.htm) element for each column and has no attributes. If no filters are predefined by <columnFilter> all determined values are displayed in the table. Allowed sub-elements | Quantity  
---|---  
[columnFilter](refformui_r_columnfilter.htm) | any
