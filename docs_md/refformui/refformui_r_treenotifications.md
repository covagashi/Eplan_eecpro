# treeNotifications

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  treeNotifications The <treeNotifications> element is only implemented in a tree structure when a trigger event occurs that is supposed to execute an action, e.g. when a user clicks a project component.
