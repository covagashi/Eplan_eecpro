## Inserting pages and fragments

Pages and fragments are added via the popup menu of the corresponding object in the project or via Drag & Drop directly from the library. The following graphic shows how to add a page via the popup menu:

The Page classes Finder dialog opens.

In this example, we choose the Lights page. This inserts the Lights page into the schematic.

Lamps can also be added on this page.

Lamps are added, like pages, via the popup menu or via Drag & Drop.
