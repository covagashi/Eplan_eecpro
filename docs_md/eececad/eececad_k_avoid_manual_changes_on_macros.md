## Preventing manual changes to macros

The Edit Macros Only Externally model variable prevents manual changes to ECAD macros from within EEC.

If the value of the model variables is set to true, the menu entry remains visible, but is grayed out (see the figure below). This serves as a note to the user that the function has not been removed generally, but has simply been disabled.
