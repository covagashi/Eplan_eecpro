## Creating a chapter

Objects of the **Chapter** type are used to create a structure within a schematic into which the pages will be placed. This way, the page macros can be arranged according to the Eplan Electric P8 tree structure, and page numbers assigned according to specific criteria, so that, for example, the pages for sensors start at the page number 200.

Unlike all other ECAD objects, Chapter macros have neither internal nor external resources.

An object of the Chapter type is generated via the popup menu New > ECAD > Chapter.
