## Basic wiring diagram with fuse and contact

The basic wiring diagram is created by removing the individual lamps. A placeholder object is added to define the required parameters, as well as the sockets of the possible number of lamps.

This basic wiring diagram is saved as a window macro in the folder defined for the resources.
