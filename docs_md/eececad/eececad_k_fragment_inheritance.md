## Inheritance of fragments

If several fragments have the same basic properties, and if there are rules (formulas) that are common to all fragments, it is often useful to combine the common elements in an abstract fragment.

An abstract fragment thus defines a classification level from which specific fragments of the modular system are derived. The fragments thus derived are assigned the basic properties defined for the abstract fragment automatically.

A Fragment object that is set to Abstract on the Attributes page can also be saved without a resource.
