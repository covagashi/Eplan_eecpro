# Inserting schematic objects via Finder

This functionality is only available for certain module packages. [Info / Copyright](license.htm)

You are here: 

## Inserting schematic objects via Finder

Requirement: An empty project has already been created, and the library with ECAD objects has been imported.

Using the popup menu of the empty project, select the menu item Add > ECAD > Schematic:

The Finder dialog for wiring diagram classes appears. The desired objects can be located in the Finder and inserted:

Use the [Search] button to display the available wiring diagrams. Confirm your selection with [OK] to create a root of the "ECAD" discipline as well as an instance of the selected schematic in the project.
