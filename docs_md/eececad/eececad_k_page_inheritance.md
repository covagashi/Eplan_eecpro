## Inheritance of pages

If several pages have the same basic properties, and if there are rules (formulas) that are common to all pages, it is often useful to combine the common elements in an abstract page.

An abstract page thus defines a classification level from which specific pages of the modular system are derived. The derived pages are assigned the basic properties defined for the abstract page automatically.

A Page object that is set to Abstract on the Attributes page can be saved also without a resource.
