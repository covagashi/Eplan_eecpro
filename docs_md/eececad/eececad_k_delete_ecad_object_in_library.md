# Deleting ECAD objects from the library

This functionality is only available for certain module packages. [Info / Copyright](license.htm)

You are here: 

## Deleting ECAD objects from the library

The library object can be deleted using its context menu. In the process, the corresponding library object and all project instances of the object are deleted.

A prompt must be confirmed to go ahead with the deletion.
