## Editing macros directly from within EEC

Regardless of whether the content of a macro is stored externally or internally, the macro can be opened directly from within EEC for editing. To do this, use the menu item Edit Macro in the popup menu of the respective page or fragment respectively.

The macro is loaded directly in Eplan Electric P8 and can also be edited there. After editing, the edited temporary page in the graphical editor must not be closed. To save the changes in the macro, Eplan Electric P8 **must** be closed. This will save the changes directly to the resource.

If parameters were added, changed or deleted during editing, the synchronization dialog appears after closing Eplan Electric P8. In the following example, two new parameters PageName and PageDescription have been added within the Lamp macro. In the dialog, the parameters before and after the change are contrasted in two columns.

Saving the editor applies the changed parameter interface of the macro. To manually add any parameters that may have been omitted, use [>>>] between the two windows. This way, even a parameter not used in the parameter interface of the macro can be integrated into the EEC object.
