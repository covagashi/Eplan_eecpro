# IMX import

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  IMX import The purpose of the IMX import is to create projects based on a modular system. The import of the IMX files commonly occurs through the import wizard, or an equivalent command. The XML file to be imported must meet the usual requirements for IMX files.
