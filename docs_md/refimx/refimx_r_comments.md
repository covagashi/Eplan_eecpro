# Comments

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Comments In large IMX files, it is helpful to document parts with comments. Comments always start with <!-- and end with \-->. In between may lie a few comments, but also entire sections. [Example code for a comment line](javascript:void\(0\);)
    
        <!-- This is a comment line -->

[Example code for a commented-out program block](javascript:void\(0\);)
