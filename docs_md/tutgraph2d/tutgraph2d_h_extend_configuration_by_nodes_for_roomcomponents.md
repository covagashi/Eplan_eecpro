# Expanding the configuration with nodes for room components

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Expanding the configuration with nodes for room components The nodes for room components must be configured in such a way that the corresponding image, suitable for the type of room component, is inserted in the diagram. To that end, there are images in an image register assigned to the types of room components.
