# Test 3 - Inserting the room components in the diagram

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Test 3 - Inserting the room components in the diagram A check should be carried out whether the project components Worker, Telephone, PC, Printer, 2wayLAN_Jack and 5waySwitch can be dragged from the project tree into the rooms of the diagram.
    1. In the Project Catalog, open the diagram IT_SolutionMap with a double-click.
    1. Add the project component Worker by Drag & Drop from the project tree in the Room A 4.1 Office.
Upon releasing the mouse button, the image for a worker is inserted.
    1. Repeat steps 2 and 3 with all the project components that are arranged below Room, Room2, Room3 and Room4 in the project tree.
    2. Save the diagram ([Ctrl] \+ [S]).
