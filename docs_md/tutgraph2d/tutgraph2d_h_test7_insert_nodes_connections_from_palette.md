# Test 7 - Insertion of nodes and connections from the palette

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Test 7 - Insertion of nodes and connections from the palette A check should be made whether the palette entries for inserting nodes and connections in the diagram work.
    1. In the Project Catalog, open the diagram IT_SolutionMap with a double-click.
    1. Select the palette entry Floor.
    2. Navigate the mouse pointer without holding down any mouse button to a free area of the diagram.
    3. Save the node Floor with a left click.
    4. Repeat the insertion with the palette entries for Room, PC and 2way LAN Jack and Worker
    1. Select the palette entry Worker connection.
    2. In the newly inserted room click first on Worker and then on PC.
    3. Select the palette entry LAN connection.
    4. Click in succession on the slave node of the PC and then on a slave node of 2wayLANJack.
    5. Close the diagram IT_SolutionMap without saving the changes.
