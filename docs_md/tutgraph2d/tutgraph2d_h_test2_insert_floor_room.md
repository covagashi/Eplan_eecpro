# Test 2 - Inserting Floor and Room in the diagram.

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Test 2 - Inserting Floor and Room in the diagram. A check should be made whether the component Floor can be dragged from the project tree into the diagram.
    1. In the Project Catalog, open the diagram IT_SolutionMap with a double-click.
    1. Drag the project component Floor with the mouse button held down from the project tree to the diagram editor area.
Till the mouse button is released, a rectangle of the size of the node to be added is displayed.
    1. Release the mouse button over the diagram editor.
Upon releasing the mouse button, a rectangle with a dash-dotted line and the name Floor A 4 is added.
    1. Repeat steps 2 and 3 with the project components Room, Room2, Room3 and Room4.
    2. Save the diagram ([Ctrl] \+ [S]).
To enlarge the visible editor area, the palette can be closed with a click on the triangle next to Palette and a corresponding zoom factor can be selected in the toolbar.
