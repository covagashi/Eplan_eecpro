# Checking moving the nodes

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Checking moving the nodes A check should be carried out whether the nodes for Room, and all the devices and workers can be moved from one parent node to another.
    1. In the Project Catalog, open the diagram IT_SolutionMap with a double-click.
    2. Create an additional floor (Floor2).
    3. Open the Form-UI for the new floor and enter the value A 5 into the Floor number field.
    4. Save the diagram ([Ctrl] \+ [S].
    1. Drag Room3 with the left mouse button held down from Floor to Floor2.
Note If moving a node is not possible, the mouse pointer symbol changes into a prohibitory notice.
    1. In Floor2, create an additional room (Room5).
    2. Open the Form-UI for the new room and select the value Office in the Room type field.
    3. Save the diagram ([Ctrl] \+ [S].
    4. Hold down the [shift] key and select all of the nodes in Room3.
    5. Drag the marked nodes with the left mouse button held down from Room3 to Room5.
