## Parameter sequences and control structures

Control structures and parameter sequences that play a decisive role in generation should be stored in the Word document, which in turn is stored in a main-document or fragment-document object. Whereas the parameter sequences are used to insert the value of the parameter in place of the sequence, control structures provide the generator with all of the information required to insert sub-elements from the discipline structure.
