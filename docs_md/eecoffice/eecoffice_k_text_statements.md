## Control structures in the continuous text

Much like parameter sequences, control structures, too, are defined by a predetermined syntax and exert a considerable influence on the generation process. All control structures begin with the token (*{ and end with the token }*). The control structure, including the required arguments, is contained between the tokens. Depending on whether the goal is to create a table or a continuous text, the control structures behave differently, i.e. their use entails different requirements. The following control structures are supported in Word:

     * [IF](eecoffice_k_text_statements_if.htm)
     * [LOOP](eecoffice_k_text_statements_loop.htm)
     * [IncludeDocument](eecoffice_k_text_statements_includedocument.htm)
