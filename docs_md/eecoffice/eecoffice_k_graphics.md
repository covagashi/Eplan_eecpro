## Images

In general images of the following formats are supported:

     * DIB
     * EMF
     * GIF
     * JPEG
     * PICT
     * PNG
     * WMF

Images can be directly content of a main document or of a fragment. When generating the project document directly, inserted images are applied 1:1 (see [Internal images](eecoffice_k_graphics_internal.htm)).

Images can be stored on the file system (called external images). External images can be inserted into the document by means of a parameter sequence (see [External images](eecoffice_k_graphics_external.htm)).

Images can be inserted into a document via image objects (see [Image objects](eecbase_k_mediasource_image_object.htm)).
