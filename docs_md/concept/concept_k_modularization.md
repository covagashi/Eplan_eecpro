## Modularization

The basic approach underlying the creation of modular systems consists in modularizing existing components and expanding them to include necessary variants.

The objective is to create new variants from multiple universally available objects. This requires mechanisms capable of combining discipline-specific objects properly. These mechanisms are implemented differently depending on the discipline.
