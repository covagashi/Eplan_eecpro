## Configuring virtual mechatronic components

The goal of functional mechatronical components is to achieve **functional configuration** of machines and plants despite the fact that project documents are structured differently. The required approach is illustrated in the following diagram.

Project managers configure a machine with the aid of functional mechatronical components and their parameter values.

The functional mechatronics components of the modular system have references to components contained within the different disciplines.

The structures of each project document are built from the mechatronical configuration via these references.

If there are unambiguous rules for building discipline-specific structures, these structures can be created automatically by **structure generators**. The amount of effort that goes into the project can thus be significantly reduced if a modular system stipulates which project documents are to be created and if these rules can be represented by structure generators.
