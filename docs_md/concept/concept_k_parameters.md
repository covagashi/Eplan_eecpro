## Parameter

The interface of a component is defined by **parameters**. Each parameter has a name, a type, a value, and a comment.

Examples of different types of parameters:

Name | Value | Type | Comment  
---|---|---|---  
ToolchangerAvailable | true | Boolean | Toolchanger available?  
AmountGripper | 5 | Integer | Number of installed grippers  
Safety factor | 1.50 | Real | Safety factor protecting against disconnection  
DT | =ZS+S1-M01 | String | Device tag
