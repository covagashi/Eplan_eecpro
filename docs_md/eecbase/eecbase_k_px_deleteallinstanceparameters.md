## deleteAllInstanceParameters

Deletes all parameters of the project component with the absolute path.

deleteAllInstanceParameters(String instanceAbsPath)  
---  
Argument | String | instanceAbsPath | Absolute path to the project component.  
Return value | Boolean | true = all parameters have been deleted successfully.  
false = no parameter has been deleted.
