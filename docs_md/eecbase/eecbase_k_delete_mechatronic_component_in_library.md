# Deleting mechatronic components in a library

This functionality is only available for certain module packages. [Info / Copyright](license.htm)

You are here: 

## Deleting mechatronic components in a library

They can be deleted via the context menu of the library component. In the process, the corresponding library object and all project components of this type are deleted.

If one or several objects are to be deleted from the library catalog, a prompt will be appear where the deletion of individual objects can be confirmed and / or canceled.
