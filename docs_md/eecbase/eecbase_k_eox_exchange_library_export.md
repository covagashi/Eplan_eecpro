## Exporting libraries/projects as an EOX file

To export libraries or model data both an export wizard of the EEC user interface and a command can be used (see [Engineering.ExportEOXCommand](refcommands_r_engineering_exporteoxcommand.htm)).

How to export model data as an EOX file by means of the export wizard of the EEC user interface:

    1. Click File > Export....

The Export wizard opens.

    1. Select Model > Wizard for exporting model data in the EOX format.
    2. Click [Next >].

The wizard for EOX export starts.

    1. In the EOX file field, enter a file name directly, or use [Browse...] to select a specific storage location.
    2. Highlight the libraries/projects to be exported.

Selected libraries/projects are identified with a check mark , while the dependent libraries/projects are identified by a square .

    1. Click [Finish].
