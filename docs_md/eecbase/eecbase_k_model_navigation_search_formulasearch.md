## Formula search

A frequent search query refers specifically to formulas. For example, the user looks for all formulas that contain the mc command.

Beyond the text search, this search supports the more specific search for formulas:

Unlike the text search, the result table of the formula search shows the parameter (Declared In) and the component that contain the formula:

The result is displayed in a table. All of the objects contained in it can be opened by double-clicking or using the Open popup menu entry. Marked objects are deleted from the result list and the model via the [Del] key or the popup menu entry Remove Selected Match. All of the matches are removed from the result list and the model with the Remove All Matches popup menu entry.

The Search/replace... function can be selected from the popup menu.

The Search/replace function encompasses the usual possibilities for carrying out changes in the currently determined formulas.
