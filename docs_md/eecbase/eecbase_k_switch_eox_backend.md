# Changing the EOX back-end

This functionality is only available for certain module packages. [Info / Copyright](license.htm)

You are here: 

## Changing the EOX back-end

After the first start with selection of a EOX file for data management, EEC always starts with this EOX file.

The menu File > Switch back-end can be used both to open the dialog for selecting an EOX file as well as a different EOX file that has already been used.

    1. Click File > Switch back-end.
    2. Mark a previously selected EOX file.

The data model is immediately changed to the selected EOX file.

    3. Click Select.

The dialog Choose new backend opens.

    4. Select a EOX file
    5. Confirm your selection with [OK].

The selected file appears in the list of previously selected EOX files at the next time the data basis should be switched.
