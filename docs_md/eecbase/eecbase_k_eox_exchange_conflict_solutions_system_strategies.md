## System strategy

The system strategy Select value can be applied to conflicts that can be solved by overwriting the components of the internal model with the components of the EOX file or vice versa.

The following image shows the application of the system strategy Select value (see [Importing libraries/projects as an EOX file](eecbase_k_eox_exchange_library_import.htm) Scenario 2).

The conflict can be solved via the system strategy Select value. The user must select whether the target value is to be the value from the EOX file or the value of the internal model.

If this strategy can be applied to all other conflicts as well, it is possible to solve all conflicts of the same type without any further interaction by selecting the check box Use as global strategy.

This, then, leaves the conflicts (if any) that can be solved only through a manual strategy.
