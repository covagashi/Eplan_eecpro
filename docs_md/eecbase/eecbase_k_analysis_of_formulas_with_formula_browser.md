## Analysis of formulas using the Formula browser

Apart from the Formula Inspector, the Formula browser shows details on formulas of objects in the project tree or the open editor.

Primary the formula browser shows all parameters of an object, which is selected in the project tree (1) or of an opened editor with their formulas in a table to obtain an overview. The table shows the information Name, Data Type, Result, Formula, and Formula Origin in columns of the same designations. The formulas cannot be edited in this table, but the table is always synchronized with the content of a selected editor when the synchronize button is pressed, which means that changes to a formula in the selected editor are immediately overtaken without needing to save the editor.

For the parameter which is selected in the table, the formula (2) is displayed as text of graph. The representation type can be selected on the Text and Graph tabs (7).

If the representation type Graph is selected, the formula is shown as a graph model with nodes and edges. The partly result of a selected node (3) is shown on the right within a preview window (4). The tooltip shows the partly result of the node (5) where the mouse cursor is positioned.

The representation type Text shows the partly result on the right when a term of the formula is selected.

The display in the Formula browser can be extended via [Show Formula Details] and [Show Object Details]. The items Disabled and Selection formula result are additionally shown for objects.

The shortcut [Ctrl] \+ [3] gives quick access to the formula browser view.

Only one unpinned Formula browser can be displayed at any one time. In order to compare several formulas, the individual Formula browser views must be pinned.
