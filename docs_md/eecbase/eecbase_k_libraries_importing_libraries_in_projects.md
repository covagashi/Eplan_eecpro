## Importing libraries into projects

The assignment of libraries to be imported is realized on the editor page Imported libraries.

The icon  (Add library) is used to start the Library finder dialog. The libraries to be integrated can now be selected. Once the selection is complete, the relationships of the Assemblies (projects and libraries) are displayed in the Imported libraries view.

In the process, all necessary system libraries are integrated.

The  (Hiding assemblies which are not directly imported) icon allows you to only show the directly imported assemblies.
