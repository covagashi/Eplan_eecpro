## Strategies for solving conflicts

Manual and system strategies can be applied to the resolution of conflicts that may occur during synchronization.

A system strategy can be applied for all conflicts of the same type.

A manual strategy is applied only to the conflict displayed.

To reduce the number of conflicts to those that can be solved manually, one must first choose a system strategy.
