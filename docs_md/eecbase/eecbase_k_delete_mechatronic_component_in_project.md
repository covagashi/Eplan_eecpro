## Deleting mechatronic components within a project

The mechatronic component can be deleted using its context menu. It should be noted here that all sub-elements of the mechatronic object will also be deleted:

If one or several objects are to be deleted from the project catalog, a prompt will be appear where the deletion of individual objects can be confirmed and / or canceled.
