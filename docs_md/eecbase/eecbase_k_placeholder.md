## Creating placeholders

With the level configuration you can predefine hierarchies of mechatronic components. With the modular system development you can predefine the positions where the project engineer may insert further components. To this purpose a placeholder instead of a component is installed.

To create a placeholder:

    1. In the Components area click .

    1. Select the Placeholder object.

    1. Optionally change the name of the placeholder.
