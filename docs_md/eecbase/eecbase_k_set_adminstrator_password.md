## Assign administrator password

The administrator of EEC installation should define a password. To do this, he must first log on as Administrator:

The authorization is enabled if the Administrator is not the only user. Then, the login panel appears at the start.

The authorization data are also all contained in an EOX. They are ignored for synchronization purposes. During the initial import to a relational DB, they are also imported.
