## Data sources

EEC provides data source objects for access to external data sources, such as CSV, RDBMS. They provide an encapsulated abstraction for access to mostly relationally structured data.

The objective is to realize as many access instances to external data sources as possible using the default formula mechanisms. Optimization through scripts or SQL queries is still required only with complex requests or very large data volumes.
