## Management of assemblies

The content of the modular system being created is stored in a library. The objects, components and parameters of a modular system are stored in such a library.

Depending on the functional scope and desired disciplines, further system libraries have to be added. Assembly is the term given to a configuration of system, modular system and project libraries.
