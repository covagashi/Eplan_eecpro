## Creating libraries

The new library is created in the library catalog.

As a rule libraries contain specific objects, for example architecture components. Specific objects are derived from system objects. To this purpose the matching system library has to be imported into the newly created library.

To create a new library:

    1. Click  (New library).

The wizard for creating libraries opens.

    1. Enter a name for the new library.
    2. Optionally mark all system libraries.
    3. Optionally mark one or more favorite system libraries.
    4. Optionally mark one or more specific system libraries.
    5. Optionally mark all self-created libraries.
    6. Optionally mark one or more self-created libraries.

By selecting a self-created library all libraries depending on it are also marked.

    1. Click [OK].

The new library is subsequently listed in the library catalog.
