# Importing project changes

This functionality is only available for certain module packages. [Info / Copyright](license.htm)

## Importing project changes

By importing project changes (*.px file), you can extend an existing project with the contents of the project changes.

The result is saved either in the project of the same name or in a new project.

### Note

If the project change (*.px file) contains several projects, the changes cannot be saved in a new project.

### [Importing project changes into an existing project](javascript:void\(0\);)

    1. Click File > Import.
    2. The Import dialog opens.
    3. Select Project > Configuration Result (PX).
    4. Click [Next >].
    5. The Import project changes dialog opens.
    6. Select the PX file to be imported.
    7. Highlight the project whose changes you wish to import.
    8. Click Finish.
    9. The project catalog then contains the existing project extended with the imported project changes.

### [Importing project changes into a new project](javascript:void\(0\);)

    1. Click File > Import.
    2. The Import dialog opens.
    3. Select Project > Configuration Result (PX).
    4. Click [Next >].
    5. The Import project changes dialog opens.
    6. Select the PX file to be imported.
    7. Select the option Create project.
    8. Enter a name for the new project.
    9. Click Finish.
    10. The project catalog then contains the new project.
