## Nesting of mechatronic components

Generally, there are recurring configurations in a project. To minimize the configuration work, closed units can be formed. These consist of a component in which further components are installed (pre-initiated). By means of Disablers the installed components can be deactivated, depending on the type of the project.

Mechatronical components have the Components field below the editor menu.

With  (Add component) a component, a placeholder or an extension point is added.

With  (Duplicate component) a selected installed component is duplicated.

With  (Replace component) a selected installed component is replaced with another component.
