## Importing libraries

Libraries can import further libraries. This way, the necessary libraries for a modular system can be selected according to the task at hand. To do this, open the library and, using the  button, open the Imported Libraries editor page.

The icon  (Add library) is used to start the Library finder dialog. The libraries to be integrated can now be selected. Once the selection is complete, the relations between the libraries are represented graphically.
