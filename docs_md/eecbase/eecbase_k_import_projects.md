## Importing projects and components

It is possible to create new projects or to insert components into existing projects by using commands and an import wizard for different file formats.

In addition parameter values can be set in the imported projects and components.

The commands are available for importing:

     * [Engineering.ForcefulEOXImportCommand](refcommands_r_engineering_forcefuleoximportcommand.htm)
     * [Engineering.ImportCSVCommand](refcommands_r_engineering_importcsvcommand.htm)
     * [Engineering.ImportExcelCommand](refcommands_r_engineering_importexcelcommand.htm)
     * [Engineering.ImportIMXCommand](refcommands_r_engineering_importimxcommand.htm)
     * [Engineering.ImportIMXWithTemplateCommand](refcommands_r_engineering_importimxwithtemplatecommand.htm)
     * [Engineering.ImportPXCommand](refcommands_r_engineering_importpxcommand.htm)
     * [Engineering.ImportLibrariesCommand](refcommands_r_engineering_importlibrariescommand.htm)
     * [Engineering.ImportXMLCommand](refcommands_r_engineering_importxmlcommand.htm)
