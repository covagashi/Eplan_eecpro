# Exporting plug-ins

This functionality is only available for certain module packages. [Info / Copyright](license.htm)

## Exporting plug-ins

The newly created plug-in must at first be exported, and then copied to the folder \dropins.

    1. Highlight the plug-in (for example, de.eplan.px.update) in the Package Explorer.
    2. Right-click to open the context menu.
    3. Select Export in the context menu.

The export wizard opens.

    4. Open the Plug-in Development branch.
    5. Select Deployable plug-ins and fragments.
    6. Confirm with [Next >].

The next wizard dialog opens.

    7. The correct plug-in must be highlighted in the Available Plug-ins and Fragments list.
    8. On the Destination tab, choose as the output directory the dropins directory of the current EEC installation.
    9. Close the wizard with [Finish].

A message window shows the export of the plug-in by way of a progress bar.
