## System libraries

System libraries contain specific objects from which objects are derived into a new library.

The following libraries are available:

System library | Description  
---|---  
AutomationWorX | Specific PLC objects for AutomationWorX  
CoDeSys | Specific PLC objects for CoDeSys  
DataSource | Objects for data sources  
Document | Objects for documents  
ECAD | Objects for schematics  
ECAD3D | Objects for 3D panel layouts  
Engineering | Base objects for mechatronics modular systems  
Graph2D | Objects for the graphical configuration  
JobServer | Objects for Job Server  
MediaSource | Objects for images and their management  
Office | Generally valid objects for Microsoft Office documents  
PLC | Generally valid PLC objects  
STEP 7 | Specific PLC objects for STEP 7  
TextDiscipline | Objects for text documents  
Word | Objects for Microsoft Word documents
