## Creating Projects

New assemblies, i.e., libraries or projects, are created in the library and / or project catalog.

Right-click to open the popup menu in the project catalog. Here, select New > Project. Alternatively, it is also possible to use the corresponding symbol: 

The project wizard appears for the creation of a new project. In the wizard, it is possible to select the name of the project and the libraries to be imported, etc. Alternatively, one can also define new projects using the project editor.

The project editor has three entries in its editor menu. By default, the Attributes editor page is open. Here, the name of the project must be entered. This can be chosen freely, but there must not be any duplicates. Each project must have a unique name. The name is applied by saving the editor.
