## OpenOfficeBaseDatasource

OpenOfficeBaseDatasource is available for access to OpenOffice Base (odb files):

### Note

OpenOffice Base behaves exactly like hsqldb in the case of SQL requests.

When editing OpenOfficeBaseDatasource, it must be ensured that the OpenOffice Quickstarter is closed. Just to be sure, there will be a prompt prior to editing.

The display of the warning can be activated/deactivated in the settings.
