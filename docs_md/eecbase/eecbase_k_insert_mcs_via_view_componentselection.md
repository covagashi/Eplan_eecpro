## Inserting via the Component Selector view

In the Component Selector view, all components (2) are listed that are derived from an abstract library component (1). If the abstract component has parameters, their values (3) will be shown to the right of the names of the derived components. The selection of components can be limited by enabling the filter function (4). An input line appears (5) where one filter value per column can be entered. Using Drag & Drop, the listed components can be integrated into the project structure (6) in the Project Explorer as well as into a Form-UI (7).
