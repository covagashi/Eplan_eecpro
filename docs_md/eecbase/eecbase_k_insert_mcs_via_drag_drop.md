## Inserting via Drag & Drop

Mechatronic components are inserted in the project via Drag & Drop from the library, depending on the initial situation, by placing the component either on the project, the root of the mechatronic components or a placeholder.

### Note

The selection formula for placeholders and extension points is calculated only for the selection dialog, but not during the generation and not during the instantiation via Drag & Drop. If the instantiated component contains further extension points with selection formulas, these will be evaluated.
