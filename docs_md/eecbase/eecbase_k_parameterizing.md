## Parameterization

Mechatronic components have properties. Examples of this are Power, Revolutions, Variant xy available, DT or PLC address. Parameters are used to describe these properties of a component. Parameters are used to realize specific variants of components. Parameters are used in a large number of components with the same meaning.
