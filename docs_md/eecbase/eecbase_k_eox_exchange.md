## Data exchange via EOX

EOX files (Engineering Object Exchange Format) offer the possibility to exchange any libraries/projects (assemblies).

For the development of models it is possible to exchange only the libraries by means of EOX files. The export of project modifications in PX files should be used primarily for the archiving of projects.
