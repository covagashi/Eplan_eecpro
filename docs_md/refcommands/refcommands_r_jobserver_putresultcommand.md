# JobServer.PutResultCommand

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  JobServer.PutResultCommand JobServer.PutResultCommand Only available if the jobserver.worker plug-in is installed (see [Job Server](admin_r_installation_variant_job_server.htm)). Sets the value of a key-value pair in the ResultData object. The ResultData object is returned in the result of the service query. Existing values of the ResultData object are overwritten. Argument | Type | Description  
---|---|---  
key | String | key  
data | Object | Value
