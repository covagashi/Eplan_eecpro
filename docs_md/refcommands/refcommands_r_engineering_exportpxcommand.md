# Engineering.ExportPXCommand

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Engineering.ExportPXCommand Engineering.ExportPXCommand Exports changes to the project to a PX file. Argument | Type | Description  
---|---|---  
projectName | String | Name of the project to export  
absolutePXFilePath | String | Absolute name of the PX file to export to  
withExtensions | Boolean |  Option indicating whether objects positioned on extension points should also be exported: true = export objects that are positioned on extension points false = do not export objects that are positioned on extension points
