# JobServer.SetProgressMessageCommand

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  JobServer.SetProgressMessageCommand JobServer.SetProgressMessageCommand Only available if the jobserver.worker plug-in is installed (see [Job Server](admin_r_installation_variant_job_server.htm)). Sets the value that is returned in the result of a status query for the "message" key. Argument | Type | Description  
---|---|---  
message | String | Message text that is to be displayed during the Job execution.
