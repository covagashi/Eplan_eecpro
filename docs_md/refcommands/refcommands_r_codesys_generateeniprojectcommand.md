# CoDeSys.GenerateENIProjectCommand

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  CoDeSys.GenerateENIProjectCommand CoDeSys.GenerateENIProjectCommand Passes the created CoDeSys program code to the ENI server. This requires that an ENI Server must be reachable. The basic properties of the ENI server can be set in the model variables (Model > Model variables > Disciplines > PLC > CoDeSys > Code generator). Argument | Type | Description  
---|---|---  
obj | Root | Root object of the discipline
