# Commands

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Commands Commands are objects, that provide algorithms for certain tasks (e.g. creating of a schematic diagram). The algorithms are implemented in execute methods. Each command has at least one execute method. Several execute methods with different arguments can also be specified. Commands are connected to the user interface within actions (calls by the right mouse key or Form-UI (see also [Commands and actions](concept_k_commands_and_actions.htm)). The plug-ins of the system provide commands via system libraries.
