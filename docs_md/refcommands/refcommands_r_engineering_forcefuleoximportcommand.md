# Engineering.ForcefulEOXImportCommand

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Engineering.ForcefulEOXImportCommand Engineering.ForcefulEOXImportCommand Imports an EOX file. The EOX file contains all information necessary for the project import. Argument |  Type |  Description  
---|---|---  
pathToEOX |  String |  Path to the EOX file.  
The EOX file contains all the information necessary for the project import but not the object on which the import has to be performed. Argument |  Type |  Description  
---|---|---  
object |  Object |  Project or project component.  
pathToEOX |  String |  Path to the EOX file.
