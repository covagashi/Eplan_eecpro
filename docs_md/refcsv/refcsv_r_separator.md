# Separator

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Separator By default the ImportCSVCommand command interprets a semicolon (;) as the separator. The CSV file can contain any character as the separator. This then has to be specified as an argument for the command (see [Engineering.ImportCSVCommand](refcommands_r_engineering_importcsvcommand.htm).
