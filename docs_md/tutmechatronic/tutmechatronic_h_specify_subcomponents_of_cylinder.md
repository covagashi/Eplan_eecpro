# Specify subcomponents of the Cylinder component

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Specify subcomponents of the Cylinder component The Cylinder component must be supplemented with two valves.
    1. Assign the Valve component twice to the Cylinder.
    2. Name the installed components Valve_1 and Valve_2.
