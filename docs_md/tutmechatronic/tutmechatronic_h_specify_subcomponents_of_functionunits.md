# Specify subcomponents of function units

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Specify subcomponents of function units Like the Functiongroups, the Functionunits, too, must be supplemented to include Actuators and Sensors.
    1. Assign to Axis the following components:
       * Cylinder
       * Valve
       * Rename Positionsensor_inductive to Position_1
       * Rename Positionsensor_inductive to Position_2
       * Rename Positionsensor_inductive to Position_3
       * Rename Positionsensor_inductive to Position_4
    2. Assign to Gripper the following components:
       * Valve
       * Pressuresensor
    3. Assign to Orientationinspector and Stack each the Positionsensor_optical component.
    4. Assign to Separator the following components:
       * Cylinder
       * Valve
       * Rename Positionsensor_inductive to Position_1
       * Rename Positionsensor_inductive to Position_2
