# Create unit for parameter

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Create unit for parameter Like component specifications, parameters are individual objects. For the parameter now required, the Parameter unit is created:
    1. Select New > Engineering > Unit in T_Mechatronic_ModularSystem in the popup menu of the Mechatronics unit.
    2. In the Name field, enter Parameter.
