# Create new project

This functionality is only available for certain module packages. [Info / Copyright](license.htm) You are here:  Create new project Use the project wizard to create a new project.
    1. Click  in the Project Catalog view.
    1. Enter the name Feeder.
    2. Click [Next >].
    1. Select the check box for the library to be imported, T_Mechatronic_ModularSystem.
Tip Manually selected libraries are identified by a check mark. Libraries imported due to dependencies are marked with a square.
    1. Click [Next >].
    1. Highlight the Feeder level component.
    2. Close the project wizard by clicking [Finish].
The result of the project wizard is as follows: Note So far, you have only worked with libraries. Libraries, object-oriented terms, contain Classes. Symbols of objects of the class level (libraries, units, etc.) are shown in blue. The newly created project, in object-oriented terms, is located at the instance level. Objects of this level are shown in orange.
