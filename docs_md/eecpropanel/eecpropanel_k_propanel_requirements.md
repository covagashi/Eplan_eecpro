## Requirements for the ECAD3D discipline

The ECAD3D discipline is used to create components for the mechanical layout of enclosures or parts of enclosures as well as their placement in a layout space.

Requirement:

     * The ECAD3D license module is part of the current installation.
     * A valid version of Eplan Electric P8 with the extension for Pro Panel is installed.
     * For the setting Disciplines > ECAD > Eplan Electric > Macro project template a project template (*.zw9) is specified for which the macro project is selected as the Type of project.
     * For the ECAD component of the WiringDiagram type, a basic project template (*.zw9) is specified that corresponds to the current version of Eplan Electric P8, for example IEC_bas001.zw9.

In order to edit the discipline ECAD3D you require the ECAD3D system library that is to be imported into the current modular system library. This system library contains the discipline-specific basic classes and methods for Eplan Pro Panel.
