## Macro variants

EEC supports the use of Pro Panel macro variants.

A macro variant is specified by means of a parameter for Pro Panel components. The parameter name is agreed analogously to the P8 discipline via the model variable Parameter Name For The Window Macro Variant.

The specification of the variant may be any string, but it must correspond to and existing variant in order not to cause an error.

The default variant of a Pro Panel macro variant is specified with 1, but can be replaced subsequently by any string.

In order to use the default variant either no value is specified for the parameter or the Pro Panel component is defined without the parameter.
